## 概要

骨原発悪性リンパ腫（primary bone lymphoma; PBL）は、骨に原発し全身リンパ腫の骨病変とは独立して診断される稀な悪性腫瘍である。骨原発悪性腫瘍の約5〜7%を占める。組織学的には大細胞型B細胞リンパ腫（DLBCL）が約80〜90%を占める。

**骨原発の定義**（全身リンパ腫との区別のため）：
- 孤立性骨病変
- 所属リンパ節以外のリンパ節腫大がない
- 全身リンパ腫の確定から6か月以上を経過した後の骨発症（一部の定義）

- **好発年齢**：40〜60歳代（骨腫瘍の中では比較的高齢）
- **好発部位**：大腿骨・脛骨・骨盤・脊椎・上腕骨
- **臨床的特徴**：骨破壊の程度に不釣り合いな巨大軟部腫瘤が特徴的。局所疼痛・腫脹。全身症状（B症状）を伴うことあり

## 画像所見

骨原発リンパ腫の最大の画像的特徴は「**X線での骨破壊が軽微な割に、MRIでの骨髄置換・軟部腫瘤が著明**」な点である。<Cite id={1} />

### 単純X線写真

| 所見 | 特徴 |
|------|------|
| 辺縁 | 不明瞭（Permeative〜Moth-eaten型）〜一見正常にも見える |
| 骨破壊 | 浸潤性で微細（虫食い状〜透明性変化）。X線の感度は低い |
| 骨皮質変化 | 軽度の内骨膜侵食〜正常（初期段階ではほぼ正常に見える） |
| 骨膜反応 | 通常なし〜乏しい |
| 軟部腫瘤 | X線上は直接描出されないが、骨破壊と不釣り合いに大きい |

### CT

CTは骨皮質内の微細な変化（虫食い状侵食・internal cortical scalloping）を検出し、軟部腫瘤の範囲と造影効果を評価する。均一な軟部腫瘤内の壊死・石灰化は稀である。

### MRI

MRIは骨原発リンパ腫の評価において最も重要なモダリティである。<Cite id={1} />

| シーケンス | 所見 |
|-----------|------|
| T1強調像 | 広範な低信号（骨髄置換）。骨髄内進展はX線で評価困難な早期から明瞭 |
| T2強調像/STIR | 中〜高信号。均一〜軽度不均一 |
| 造影T1 | 均一〜不均一な増強。壊死は少ない（均一な増強が多い） |
| DWI | 高いb値（b=800〜1000）でも拡散制限を認める（ADC低値）。リンパ腫の細胞密度の高さを反映 |

**DWIでの著明な拡散制限**は骨原発リンパ腫に比較的特徴的であり、他の骨腫瘍との鑑別に有用な所見である。

### FDG-PET/CT

FDG-PET/CTは骨原発リンパ腫の診断・ステージング・治療効果判定に極めて有用である。DLBCLは高度FDG集積腫瘍であり、全身の病変検索に最も優れる。骨病変はMRI単独では評価困難な多発病変を一度に評価できる。

## 鑑別診断

| 疾患 | 鑑別のポイント |
|------|---------------|
| **Ewing肉腫** | 若年者（25歳未満）。骨幹部中心。onion skin骨膜反応。EWSR1転座（FISH/PCR）。MRI所見類似 |
| **転移性骨腫瘍** | 既知の原発巣。多発。リンパ腫より骨破壊が明確なことが多い |
| **多発性骨髄腫** | 高齢者。多発性打ち抜き像。M蛋白・骨髄検査で確定 |
| **骨髄炎（慢性）** | 軟部炎症の性状・分布が異なる。病歴・炎症マーカー。DWIでADCはリンパ腫ほど低くない傾向 |
| **线維肉腫/未分化多形性肉腫** | 高悪性度。MRI T2で高信号+壊死。骨破壊が顕著。DLBCLとのMRI所見鑑別は困難なことがある |
