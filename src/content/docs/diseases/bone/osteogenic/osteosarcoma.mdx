## 概要

骨肉腫（osteosarcoma）は、多発性骨髄腫・転移性骨腫瘍を除いた原発性悪性骨腫瘍の中で最も頻度が高く、約35%を占める。腫瘍細胞が骨様基質（osteoid）または骨を直接産生することを特徴とする。

- **好発年齢**：10〜20歳代（第一ピーク）および60歳代以降（第二ピーク：Paget病関連など）
- **好発部位**：長管骨骨幹端部（大腿骨遠位・脛骨近位・上腕骨近位で約50〜60%）
- **生物学的悪性度**：高悪性度（conventional type）が大多数

組織型別に通常型（conventional）・傍骨性（parosteal）・骨膜性（periosteal）・低悪性度中枢性（low-grade central）・二次性（secondary）等に分類され、それぞれ予後・画像所見が異なる。

## 画像所見

### 単純X線写真

通常型骨肉腫の典型的X線所見は、**骨形成と骨破壊が混在した混合型（mixed sclerotic-lytic）病変**である。<Cite id={1} />

| 所見 | 特徴 |
|------|------|
| 辺縁 | 不明瞭（Lodwick type II〜III）<Cite id={3} /> |
| 基質 | 雲状・綿花状の骨様基質石灰化（osteoid matrix） |
| 骨皮質 | 破壊・穿孔を伴うことが多い |
| 骨膜反応 | Codman三角・Sunburst pattern（放射状骨膜反応）<Cite id={4} /> |
| 軟部腫瘤 | 高頻度（70〜80%）に骨外軟部腫瘤を伴う |

傍骨性骨肉腫は骨表面に発生し、高度に骨化した充実性腫瘤として大腿骨遠位後面に好発する。悪性度は低く、骨皮質の連続性が比較的保たれる。

### CT

CTは骨皮質の破壊パターン評価と腫瘍内石灰化の検出に優れる。軟部腫瘤内の骨様基質が明確に描出でき、X線で不明瞭な場合もCTで同定可能なことが多い。肺転移の検索として、診断時に体幹CTが推奨される。

### MRI

MRIは腫瘍の骨髄内進展・軟部浸潤・神経血管束への関与・関節内浸潤評価に必須である。<Cite id={2} />

| シーケンス | 所見 |
|-----------|------|
| T1強調像 | 骨髄置換により低〜中信号。正常骨髄（高信号）との境界が進展範囲を示す |
| T2強調像/STIR | 腫瘍本体は不均一高信号。骨様基質や石灰化部位は低信号 |
| 造影T1 | 活性腫瘍部は増強。壊死部は非増強（境界が明瞭） |
| DWI | 活性腫瘍部でADC低下（拡散制限）。壊死部はADC高値 |

**スキップ病変**（skip metastasis）は同一骨内の非連続性転移巣であり、骨肉腫の約5〜10%に認められる。術前MRIでの骨全長撮影（whole-bone imaging）が推奨される。

**反応層（reactive zone）**は腫瘍辺縁外側のT2高信号帯であり、腫瘍細胞を含まない反応性浮腫を示す。切除縁設定においてMRI所見の解釈が重要である。

## 鑑別診断

| 疾患 | 鑑別のポイント |
|------|---------------|
| **Ewing肉腫** | 骨幹部発生が多い。骨形成所見に乏しい。onion skin骨膜反応。軟部腫瘤が大きい |
| **転移性骨腫瘍** | 中高年。多発。原発巣の既往。椎体・骨盤・肋骨など体幹骨に多い |
| **骨髄炎** | 発熱・炎症反応。骨端板をまたぐ骨破壊・周囲軟部炎症。sequestrum（死骨）を伴うことあり |
| **線維性骨異形成** | 磨りガラス様（ground-glass）基質。骨形成所見なし。骨変形（shepherd's crook変形） |
| **傍骨性骨肉腫** | 骨表面発生。高度骨化。悪性度低い。大腿骨遠位後面に好発 |
