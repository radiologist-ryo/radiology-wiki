## 概要

硬膜動静脈瘻（dural arteriovenous fistula; **dAVF**）は、硬膜動脈が流入し短絡部が **硬膜動静脈壁およびその近傍の骨内、または架橋静脈合流部** に存在する後天的な動静脈シャント疾患である。脳動静脈奇形（bAVM）と並ぶ代表的な頭蓋内動静脈シャント疾患であるが、流入動脈の種類とシャント部位が異なる。

シャント血流による静脈うっ血、出血、正常脳実質の虚血、痙攣・意識障害などの脳機能障害を来しうるが、症状は **病変部位・シャント量・血管構築** によって多彩である。治療方針の決定には画像診断が大きな役割を担うため、放射線科医は異常所見だけでなく、病態を決定づける血管構築・分類（grading）・治療オプションを把握しておく必要がある。

## 病態

dAVF は多くは硬膜静脈洞やそこに還流する静脈に関連して発生する。流入動脈は **硬膜動脈**、流出静脈は **硬膜静脈洞または皮質静脈** となる。

### 好発部位
- 横-S状静脈洞
- 海綿静脈洞
- 上矢状静脈洞
- 小脳テント
- 顆管静脈幹部

### 症状の出方は静脈還流の状態次第
- **シャント血流が順行性に還流** → 血管雑音などの軽度症状
- **流出路の狭窄閉塞 → 静脈逆流を伴う**
  - 眼静脈への逆流: 結膜充血、眼球突出
  - 皮質静脈への逆流: 静脈性梗塞、出血（重篤）

## 分類

dAVF は静脈側の血行動態によって臨床像と治療方針が大きく左右されるため、分類はその違いに基づいて定義される。

### Cognard 分類

臨床で最も頻用される分類<Cite id={1} />。先行する Djindjian & Merland 分類<Cite id={2} />を流出静脈の形態でさらに細分化したもの。

| Type | 血行動態 |
| --- | --- |
| I | 硬膜静脈洞に順行性に還流する |
| II | 硬膜静脈洞に還流するが、順行性血流が静脈洞狭窄・閉塞で障害され、他の静脈洞または皮質静脈への逆流を伴う |
| IIa | 硬膜静脈洞のみに逆流 |
| IIb | 皮質静脈のみに逆流 |
| IIa+b | 硬膜静脈洞と皮質静脈の両方に逆流 |
| III | 皮質静脈に直接逆流するが、皮質静脈の拡張（静脈瘤）を伴わない |
| IV | 皮質静脈に直接逆流し、皮質静脈が 5 mm 以上または正常皮質静脈径の 3 倍以上に拡張 |
| V | 頭蓋内 dAVF から **脊髄静脈への逆流** を伴う |

### Borden 分類

より簡便な分類<Cite id={3} />。Cognard 分類との対応も覚えておくと臨床で使いやすい。

| Type | 血行動態 | Cognard 対応 |
| --- | --- | --- |
| I | 静脈還流が静脈洞のみ（順行性または逆行性） | I, IIa |
| II | 静脈洞還流＋還流路の狭窄・閉塞による皮質静脈逆流 | IIb, IIa+b |
| III | 静脈洞中枢へ還流せず皮質静脈逆流のみ、または直接皮質静脈逆流 | III, IV |

## 治療

### 皮質静脈逆流なし（Cognard I/IIa, Borden I）
- 症状は拍動性耳鳴などの軽度
- **保存的治療** が選択されることも多い
- 塞栓術を行う場合は静脈洞を温存
  - 経動脈的塞栓術（TAE）
  - shunted venous pouch が存在すれば選択的な経静脈的塞栓術（TVE）<Cite id={4} />

### 皮質静脈逆流あり（Cognard IIb / IIa+b / III / IV / V, Borden II/III）
- **根治あるいは少なくとも皮質静脈逆流の消失** を目標とする<Cite id={5} />
- 罹患静脈洞へアプローチし、コイルや液体塞栓物質で TVE
- 経動脈的に罹患静脈洞や逆流皮質静脈へ液体塞栓物質を充填
- 近年: **sinus protection balloon** で静脈洞を温存しつつ、非接着性液体塞栓物質を用いた TAE で静脈洞近傍シャントを閉塞させる方法<Cite id={6} />

## 画像診断

動静脈シャント疾患の画像診断は、(a) 存在診断、(b) 血管構築把握のための精査、(c) 治療後経過観察、の 3 段階に分かれる。血管造影が血管構築把握のゴールドスタンダードだが、CTA・MRA の進歩により非侵襲的モダリティでもかなりの精度で診断可能になっている。

### 読影で押さえるべきポイント

| 着目点 | 内容 |
| --- | --- |
| 流入動脈 | 拡張・蛇行した硬膜動脈の同定 |
| 罹患静脈洞 | 異常血流信号の有無 |
| 静脈洞の順行性還流 | 保たれているか、閉塞・狭窄しているか |
| 皮質静脈逆流 | 有無を必ず評価 |
| 静脈瘤形成 | サイズと部位 |
| 周囲脳実質 | 浮腫・出血の有無 |
| 眼窩内 | 眼静脈拡張の有無 |

これらが整理できれば Cognard 分類・Borden 分類まで CT/MRI で言及可能である。

### 単純CTでのスクリーニング

- 神経脱落症状例: **脳実質の浮腫による低吸収域・出血・拡張血管** に着目
- 眼症状例: **眼静脈拡張** に留意
- 異常所見があれば造影 3D-CTA、MRI/MRA を追加検討

### 造影 CTA（3D-CTA / 4D-CTA）

- 骨近傍を走行する硬膜動脈の流入を証明する必要があり、**サブトラクション・dual-energy CT による骨除去が必須**
- 4D-CTA は時間分解能を加え血行動態評価に有用
- レポートでは: 動脈相早期での静脈描出、流入動脈、シャント部位（硬膜上 vs 脳軟膜上）、流出静脈の種類、Cognard/Borden 分類まで言及

### MRI / MRA

スクリーニングでは TOF-MRA を用いる。**静脈描出の有無と周囲拡張硬膜動脈の有無** に着目する。

#### TOF-MRA の落とし穴: 静脈洞内の異常血流信号

TOF-MRA は inflow 効果で動脈血流を描出し、静脈血流はサブトラクションパルスで抑制される。静脈内に血流信号があれば動静脈瘻を疑うが、**シャントのない正常海綿静脈洞や S 状静脈洞でも血流信号が描出されうる**。

- 原因: 腕頭静脈圧迫、静脈弁の機能不全、頭蓋外頭頸部の筋収縮による静脈血逆流<Cite id={7} />
- 鑑別: 異常信号のある静脈洞周囲に **拡張した流入動脈の血流信号があるか** で判別
- 悩ましい場合は 4D-CTA / 4D-MRA の追加を提案

#### 4D-MRA（time-resolved MRA）

血管構築・シャント血流のダイナミックな把握に有用。

- 一般に CTA より時間分解能に劣るが、**echo sharing** 技術で空間分解能を維持しつつ時間分解能を高めることが可能<Cite id={8} />
- 血管造影とほぼ同等の病変検出能が報告されている<Cite id={9} />

#### 非造影 4D-MRA（ASL ベース）

近年普及しつつある。

- **arterial spin labeling（ASL）** を応用し、近位動脈内プロトンを反転パルスで "labeling"、脳内到達タイミングでデータ収集、labeling なし画像との差分で血管データを抽出<Cite id={10} />
- **ACTRESS**（Acquisition of Control and Labeled images in the Same Shot）等の高速化技術で画質向上・撮像時間短縮を実現<Cite id={11} />
- **vessel-selective MRA**: labeling 箇所を選択して特定血管系のみ描出可能
- ASL は本来灌流画像で、シャント存在部位は **rCBF 上昇域** として描出されるためスクリーニングにも有用

### 治療後経過観察

#### CTA
- 塞栓物質（n-BCA・リピオドール混合液、コイル、Onyx™）が高吸収アーチファクトを生じる
- **metal artifact reduction（MAR）** 併用で影響を最小限に
- 術前みられた流入動脈・流出静脈の拡張蛇行の残存、4D-CTA での動脈相での流出静脈の早期描出を確認
- 静脈の還流方向の評価も重要

#### MRI / MRA
- CT より塞栓物質アーチファクトの影響が少なく、**経過観察に特に有用**
- TOF-MRA: 異常血流信号消失の有無、拡張血管の血栓化の有無、血行動態変化に伴う浮腫・出血の有無
- 4D-MRA: シャント残存・消失・再発の直接描出

## 鑑別診断

| 病態 | 鑑別ポイント |
| --- | --- |
| 脳動静脈奇形（bAVM） | 流入は皮質動脈、シャントは脳実質内・皮質、nidus 形成 |
| jugular venous reflux（正常変異） | 流入動脈の拡張なし、腕頭静脈圧迫由来の TOF 異常信号 |
| Galen 大静脈瘤 | 小児、深部静脈系の拡大 |
| 脳静脈洞血栓症 | 静脈洞内の血栓信号、シャントなし |
| 海綿静脈洞病変（炎症・腫瘍） | 海綿静脈洞拡大の所見、造影効果のパターン |
