## 概要：FLAIRの原理と SFHI とは

FLAIR（fluid-attenuated inversion recovery）は、反転回復法によって CSF（脳脊髄液）の信号を抑制しつつ T2 強調コントラストを強調する撮像法である。正常では CSF は低信号（抑制される）となり、脳実質周囲の浮腫・病変が高信号として描出される。

**Sulcal FLAIR hyperintensity（SFHI）** は、本来低信号となるべき脳溝内の CSF（くも膜下腔）が高信号を呈する所見である。本稿では非造影 FLAIR に限定して SFHI の原因と鑑別を論じる。SFHI を呈する原因は大きく①アーチファクト、②CSF 組成変化、③血行動態変化、④mass effect、⑤医原性要因に分類される<Cite id={1} />。

## SFHI を呈するアーチファクト（mimics）

アーチファクトによる SFHI は **病的変化と誤認されやすい** ため、まず除外することが重要である。

### 磁化率アーチファクト
- 金属（脳動脈瘤クリップ・補聴器など）や空気との境界部に生じる
- 信号欠損（signal void）や画像ゆがみを伴うことが多く、鑑別の手がかりとなる

### CSF フローアーチファクト
- 脳底槽・橋前槽・小脳橋角槽・Monro 孔など CSF 拍動の強い部位に生じる
- 3T 装置で顕著であり、同一スライスの CSF を横断するような「帯状」の高信号として出現することが多い

### 拍動アーチファクト（CSF・血管）
- 位相エンコード方向のゴーストとして出現
- 規則的な間隔を持つ繰り返しパターンが特徴

### 運動アーチファクト
- 上下スライスにわたる連続性がない（局在しない）
- 画像全体のボケ・ブレを伴う

## SFHI の鑑別診断

### CSF 組成変化による SFHI

CSF 中のタンパク・血液・炎症細胞・造影剤などが混入すると、FLAIR での CSF 抑制が不完全となり SFHI が生じる。

| 疾患 | 主な機序 |
|------|---------|
| くも膜下出血（SAH） | 血液混入でタンパク増加 |
| 慢性硬膜下血腫 | 隣接するくも膜下腔への影響 |
| 感染性髄膜炎（細菌性・結核性） | 炎症性タンパク・細胞増加 |
| 癌性髄膜炎（髄膜播種） | 腫瘍細胞・タンパク増加 |
| ARIA-E（アミロイド関連画像異常・浮腫型） | 血液脳関門破綻・タンパク漏出 |
| CAA 関連炎症 | 同上 |
| 軟膜黒色腫 | メラニン・タンパク増加 |

### 血行動態変化による SFHI

脳血流・静脈排出障害により脳溝内の血流停滞や側副血行路形成が生じると SFHI として観察される。

| 疾患 | 機序 |
|------|------|
| もやもや病・脳動脈狭窄閉塞 | **ivy sign**: 豊富な軟膜側副血行路 |
| 脳静脈洞・皮質静脈血栓症 | 静脈うっ滞によるくも膜下腔拡張 |
| 硬膜下血腫 | mass effect + 局所血流変化 |
| Sturge-Weber 症候群 | 軟膜毛細血管腫・静脈排出障害 |

### Mass effect による SFHI

脳腫瘍や硬膜下血腫が脳溝を圧排すると、残存する CSF が相対的に高信号に見える場合がある。

### 医原性 SFHI

- **高濃度酸素投与**: リザーバーマスク等による高流量酸素投与では、O₂による CSF 中の T1 短縮効果（酸素の常磁性）により SFHI が生じうる<Cite id={2} />。投与中止・室内気での撮像で消失する
- **ガドリニウム造影剤**: 前日までの投与によりガドリニウムが CSF 中に残留し、翌日の FLAIR で SFHI として出現することがある

## 代表症例

### 症例1：慢性硬膜下血腫 + 焦点感覚運動発作

慢性硬膜下血腫に隣接する脳溝に SFHI が認められることがある。発作との関連については、血腫隣接領域の皮質浸潤・過潅流が想定されており、ASL 灌流 MRI による血流評価で検証した報告がある<Cite id={3} /><Cite id={4} />。

### 症例2：ARIA-E（レカネマブ投与中の Alzheimer 病）

抗アミロイド抗体（レカネマブ・アデュカヌマブ）投与に関連する ARIA（アミロイド関連画像異常）のうち、**ARIA-E（浮腫/滲出）** では脳溝への FLAIR 高信号（SFHI）が特徴的な所見として現れる<Cite id={5} />。ARIA-H（微小出血・ヘモジデリン沈着）との合併にも注意が必要である。治療歴の確認と SWI/T2* による出血性変化の評価が重要である。

### 症例3：高濃度酸素投与

リザーバーマスク 10 L/min 投与中の患者の FLAIR で広範な SFHI を呈した後、室内気に切り替えて 3 日後の再撮像で消失した症例が報告されている<Cite id={6} />。酸素の常磁性による T1 短縮効果が CSF 抑制の不完全化を招く。

## 診断のポイント

1. **アーチファクト鑑別を最初に行う**: 分布・形態（帯状/焦点性）・他シーケンスとの整合性を確認
2. **臨床情報の収集**: 治療歴（レカネマブ・酸素投与・ガドリニウム投与歴）、発症経過、CSF 検査結果
3. **他シーケンスとの統合評価**: SWI/T2*（出血性変化）、DWI（梗塞・拡散制限）、ASL（血流）、T1 増強（髄膜染）
4. **病的 SFHI の可能性がある場合**: 広範・連続・臨床症状と一致する分布であれば積極的に報告する
