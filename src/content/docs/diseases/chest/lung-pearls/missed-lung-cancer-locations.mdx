## 概要

肺気腫や間質性肺炎の経過観察中、縦隔・血管周囲の特定部位に発生した肺癌を見落とすリスクがある。本症例は奇静脈陥凹に発生した LCNEC（combined large cell neuroendocrine carcinoma）であり、2年間見落とされた教訓的な症例である。

## 本症例

70歳台男性。肺気腫・間質性肺炎の経過観察中に胸部 HRCT を施行。

- **初回 HRCT**: 右下葉内側に異常なし（気腫と背側の間質影のみ指摘）
- **1年9か月後 HRCT**: 同様に右下葉内側は正常と判断
- **2年後 CT**: 右下葉内側に結節を指摘、増大経過確認後に生検で LCNEC と確定

奇静脈陥凹という読影の盲点部位に位置したため、長期間見落とされた。

## CTで見落としやすい部位

縦隔・胸膜・血管に近接した部位では、正常構造との重複により腫瘍の視認が困難になる。<Cite id={1} />

| カテゴリー | 具体的部位 |
|-----------|-----------|
| 血管と接する部位 | 奇静脈陥凹、下行大動脈周囲、肺門血管周囲 |
| 気管・気管支内 | 気管支内発育型肺癌（気管支と等吸収のため不明瞭） |
| 縦隔周囲 | 縦隔胸膜直下、心陰影後面、横隔膜直上 |
| 胸膜下 | 肺尖部胸膜下、肋骨・椎体に隣接した部位 |
| 病変内・隣接 | ブラ内・ブラ隣接部、蜂窩肺内 |

## 奇静脈食道陥凹の正常解剖

奇静脈食道陥凹（azygoesophageal recess）は、右下葉内側の肺が椎体前方で縦隔背側へ入り込んだ部分であり、奇静脈弓から横隔膜にかけて形成される。単純 X 線写真では「奇静脈食道線」として描出される。

成人では通常陥凹を形成するが、若年成人の 7〜21% では陥凹が形成されない。<Cite id={2} /> 陥凹内に位置する病変は縦隔構造との境界が不明瞭になりやすく、見落としが生じやすい。

## 下行大動脈周囲の非典型的肺癌

多発 pure GGN の経過観察中に、下行大動脈周囲に沿う細長い充実性病変として浸潤性腺癌が発見された症例が報告されている。下行大動脈周囲の血管との接触部は拡張不全による偽病変（後述）とも類似し、肺癌の指摘が困難になる。

## 見落とし防止のポイント

- **縦隔・血管・胸膜との境界を意識して読影する**: 特に奇静脈陥凹・大動脈周囲・肺尖は注意深くトレースする
- **肺野の端（縦隔側・横隔膜側・胸壁側）まで必ず確認する**
- **気管支・肺血管に沿って中枢から末梢まで追跡する**
- **経時変化の比較**: 同一部位を前回 CT と細かく比較し、わずかな変化を見逃さない
- **既知病変（気腫・間質影）に隠れた変化に注意する**
