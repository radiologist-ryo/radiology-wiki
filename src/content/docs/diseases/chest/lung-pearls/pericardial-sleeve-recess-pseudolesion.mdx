## 概要

右下肺静脈周囲には、心膜がスリーブ状に静脈を取り囲む陥凹（pericardial sleeve recess）が存在する。この陥凹に液体が貯留すると、CTで肺門周囲の軟部影として描出され、リンパ節腫大と誤認される偽病変となる。肺癌や悪性リンパ腫の術前精査・病期診断において診断的落とし穴となりうる。

## 本症例

60歳台男性。左上葉肺癌の術前精査目的で造影 CT を施行。

- **CT 所見**: 右下肺静脈周囲に血管より低吸収の軟部陰影、肺門リンパ節腫大様
- **判断**: 画像的特徴（液体吸収値、隣接肺静脈への圧排なし、心膜との連続性）から心膜スリーブ陥凹による偽病変と判定
- **臨床的意義**: N2 リンパ節腫大と誤認すると手術適応判断に影響するため重要

## 解剖学的背景

心膜は心臓・肺静脈を取り囲み、反転により複数の陥凹・洞を形成する。右肺静脈陥凹（right pulmonary vein recess）は右上肺静脈と右下肺静脈の間に位置する。肺静脈が心膜を貫通する部位では、漿膜鞘が静脈周囲をスリーブ状に包む構造をとる。<Cite id={1} />

## 画像所見

| 特徴 | 詳細 |
|------|------|
| 形状 | 典型は小さく紡錘形、右下肺静脈の前後に沿う |
| 増大時 | 液体貯留増大で上下にも拡大し卵円形 |
| 大きさ | 直径 1 cm 以上で診断上の問題となりやすい |
| 左右差 | 右側で多い（左右の肺静脈解剖学的差異による） |
| 分布 | 背側・尾側に大きい場合あり、均等でないことも |

## 鑑別と鑑別点

- **鑑別**: リンパ節腫大、心膜嚢胞
- **鑑別点**:
  - 液体の吸収値（水様〜漿液性）
  - 隣接肺静脈への圧排効果がない
  - 心膜との連続性が確認できる
  - 造影では増強を示さない

造影 CT でも判断が難しい場合、MRI（T2 強調像での液体信号）が有用なことがある。<Cite id={1} />

## 解剖学的ピットフォール

:::note[MEMO]
- **横隔膜挙上に伴う圧迫性無気肺の誤嚥性肺炎誤認**: 横断像のみでは圧迫性無気肺と気道感染の区別が難しい。冠状断像・スカウト像・胸部 X 線での横隔膜位置の確認が重要
- **右上葉切除後の中葉結節の区域誤記**: 「右肺 S2 に結節の出現」と記載されることがあるが、右上葉切除後の気管支断端を確認すれば実際は中葉の結節と分かる。区域解剖に沿った系統的な読影が誤記防止に不可欠
:::
