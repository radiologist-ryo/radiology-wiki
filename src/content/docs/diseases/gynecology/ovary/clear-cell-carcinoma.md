## 概要

卵巣明細胞癌（clear cell carcinoma; CCC）は日本で特に多く、日本における卵巣癌の約25〜30%を占める（欧米では5〜10%）。子宮内膜症からの発生が多く（約50〜70%）、類内膜癌とともに内膜症関連卵巣癌（EMOC）に分類される。プラチナ製剤への反応性が低く、HGSCと比較して化学療法抵抗性を示す。ARID1A変異が高頻度（50〜60%）に認められる。

---

## 疫学

40〜60歳代に多く発症し、日本では子宮内膜症罹患率が高い背景を反映して欧米より発生頻度が高い。大部分がStage IまたはIIで発見されるが、進行例の予後は不良である。深部静脈血栓症・肺塞栓症の合併頻度が他の卵巣癌より高く（約40%）、術前評価に注意を要する。

---

## 病理

肉眼所見では内膜症嚢胞（チョコレート嚢胞）の壁に附随する充実成分として認められることが多い。組織学的には豊富な淡明細胞質（clear cytoplasm、グリコーゲン・脂質に富む）を有する細胞からなり、管状・乳頭状・充実性の構造を呈する。ホブネイル細胞（hobnail cell）が特徴的である。

---

## 画像所見

### 超音波（US）

超音波では内膜症嚢胞内に多房性の充実成分が突出した構造が特徴的に描出される。充実成分への血流は乏しいことがある。以下に所見をまとめる。

| 所見 | 特徴 |
|------|------|
| 嚢胞 | 内膜症嚢胞（低〜中エコー）に充実成分附随 |
| 充実成分 | 扇状・ポリープ状、壁から突出 |
| 血流 | 充実成分内に血流（乏〜中等度） |

### CT

CTでは内膜症嚢胞（高CT値20〜70 HU）の壁在に造影増強を示す充実性結節が特徴的である。以下に所見をまとめる。

| 所見 | 特徴 |
|------|------|
| 嚢胞 | 高CT値の内膜症嚢胞に充実成分 |
| 充実成分 | 造影増強（++） |
| 形態 | 内膜症嚢胞＋充実成分の複合型 |
| 腹水 | 進行例で合併 |

### MRI

MRIは内膜症嚢胞と悪性充実成分の同定に最も優れる。T1高信号（内膜症成分）の嚢胞内に充実成分が描出され、DWIで拡散制限を示す。以下に所見をまとめる。

| 所見 | 特徴 |
|------|------|
| T1強調像 | 嚢胞：高信号（内膜症）＋充実成分：中等度 |
| T2強調像 | Shading sign＋充実成分（中等度信号） |
| DWI | 充実成分に拡散制限（ADC低下） |
| 造影 | 充実成分が増強 |

---

## 鑑別診断

内膜症嚢胞の壁在結節は6 mm以上・血流ありで悪性を疑う。

| 疾患 | 鑑別のポイント |
|------|---------------|
| **類内膜癌** | 内膜症由来。組織学的類似。管状腺構造。 |
| **高異型度漿液性癌（HGSC）** | 内膜症との関連少ない。腹膜播種顕著。 |
| **子宮内膜症嚢胞（良性）** | 充実成分なし。定期的超音波フォローで変化なし。 |
| **成熟奇形腫** | 脂肪・石灰化。T1高信号（脂肪）。 |
