## 概要

無性細胞腫（dysgerminoma）は卵巣胚細胞性腫瘍の中で最も多い悪性腫瘍であり、精巣の精上皮腫（seminoma）に相当する。悪性胚細胞腫瘍の30〜40%を占める。化学療法（BEP療法）および放射線療法に対する感受性が高く、悪性卵巣腫瘍の中でも予後は比較的良好である（5年生存率 > 90%）。

---

## 疫学

10〜30歳代の若い女性に好発し、全卵巣悪性腫瘍の2〜5%を占める。両側性が約10〜15%に認められ（胚細胞腫の中で最も高率）、正常な卵巣からも発生するが、性分化疾患（46,XY性腺異形成）合併例での発生リスクが高い。妊孕性温存術式の適応が積極的に検討される腫瘍の一つである。

---

## 病理

肉眼所見では充実性・淡黄色〜白色の大型腫瘤で、線維血管性隔壁が腫瘤内を分割する外観が特徴的である。組織学的には均一な大型腫瘍細胞（明るい細胞質・突出した核小体）がシート状・巣状に配列し、線維性隔壁内にリンパ球浸潤を伴う。LDH・胎盤型ALP（PLAP）が腫瘍マーカーとして上昇する。

---

## 画像所見

### 超音波（US）

超音波では充実性の均一な低エコー腫瘤として描出され、内部に線状の強エコー（線維血管性隔壁）が交差するパターンが特徴的である。以下に所見をまとめる。

| 所見 | 特徴 |
|------|------|
| 形態 | 大型充実性（多葉状） |
| 内部 | 均一低エコー |
| 隔壁 | 線状強エコー（線維血管性隔壁） |
| 血流 | 豊富 |

### CT

CTでは充実性で均一な軟部組織密度の腫瘤として描出され、内部に線維血管性隔壁が造影増強されるパターン（lobulated mass with enhancing fibrovascular septa）が特徴的である。以下に所見をまとめる。

| 所見 | 特徴 |
|------|------|
| 形態 | 大型充実性（多葉状輪郭） |
| 造影 | 均一増強、線維血管性隔壁が増強 |
| 石灰化 | まれ（点状） |
| リンパ節 | 後腹膜リンパ節腫大を伴うことあり |

### MRI

MRIでは均一なT2中等度信号の充実性腫瘤と、低T2信号の線維血管性隔壁が交差するパターンが特徴的である。以下に所見をまとめる。

| 所見 | 特徴 |
|------|------|
| T1強調像 | 均一な中等度〜低信号 |
| T2強調像 | 均一な中等度信号＋低信号の線維隔壁 |
| DWI | 著明な拡散制限（ADC著明低下） |
| 造影 | 均一増強、隔壁増強 |

## 鑑別診断

| 疾患 | 鑑別のポイント |
|------|---------------|
| **卵黄嚢腫瘍** | AFP著明高値。より不均一な充実性。若年女性。 |
| **未熟奇形腫** | 脂肪・石灰化・神経性充実成分混在。 |
| **顆粒膜細胞腫** | 中高年女性。エストロゲン産生。 |
| **転移性腫瘍** | 両側性。原発巣の検索。 |
