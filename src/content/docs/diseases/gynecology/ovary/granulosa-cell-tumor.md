## 概要

顆粒膜細胞腫（granulosa cell tumor; GCT）は卵巣性索間質性腫瘍の中で最も多く、低悪性度の悪性腫瘍である。エストロゲンを産生することが多く、不正性器出血・月経不順・閉経後出血・子宮内膜増殖症（子宮内膜癌の合併が約5〜10%）などの内分泌症状を呈する。成人型（adult type）と若年型（juvenile type）に分類され、前者が95%を占める。

---

## 疫学

成人型は50〜60歳代の閉経周辺期に多く、若年型は10〜20歳代に多い。性索間質性腫瘍の中では最も多いが、全卵巣腫瘍の1〜2%にすぎない。成人型のFOXL2変異（c.402C>G）はほぼ全例に認められ、診断的価値が高い。晩期再発（術後10〜20年以降）が特徴的であり、長期フォローが必要である。

---

## 病理

肉眼所見では充実嚢胞混合性で、自然破裂による血腹症（急性腹症）をきたすことがある。組織学的にはCall-Exner小体（顆粒膜細胞が核を囲む菊花状配列）が特徴的である。インヒビンやAMH（抗ミュラー管ホルモン）が腫瘍マーカーとして有用である。

---

## 画像所見

### 超音波（US）

超音波では充実嚢胞混合性の腫瘤として描出される。内部に出血を伴う場合は不均一な低エコーを呈する。以下に所見をまとめる。

| 所見 | 特徴 |
|------|------|
| 形態 | 充実嚢胞混合性 |
| 内部 | 不均一、海綿状（血腔形成） |
| 破裂 | 血腹症（腹腔内出血）を呈することあり |
| 内膜 | 子宮内膜肥厚（エストロゲン産生） |

### CT

CTでは充実成分と嚢胞成分が混在する腫瘤として描出される。出血を伴う場合は高吸収域を認める。子宮内膜肥厚が合併することがある。以下に所見をまとめる。

| 所見 | 特徴 |
|------|------|
| 形態 | 充実嚢胞混合性、辺縁明瞭 |
| 出血 | 高吸収域（出血成分） |
| 造影 | 充実成分が増強 |
| 子宮 | 内膜肥厚（エストロゲン効果） |

### MRI

MRIでは充実成分内に出血を示すT1高信号域を混じる「海綿状」外観が特徴的とされる（Swiss cheese appearance）。以下に所見をまとめる。

| 所見 | 特徴 |
|------|------|
| T1強調像 | 出血部：高信号（Swiss cheese pattern） |
| T2強調像 | 充実成分：中等度〜高信号（不均一） |
| DWI | 充実成分に拡散制限 |
| 造影 | 充実成分が増強 |

## 鑑別診断

| 疾患 | 鑑別のポイント |
|------|---------------|
| **類内膜癌** | 内膜症由来。内分泌症状は通常なし。 |
| **高異型度漿液性癌（HGSC）** | 腹膜播種顕著。内分泌症状なし。 |
| **線維腫・莢膜細胞腫** | 充実性。T2低信号。エストロゲン産生は莢膜細胞腫。 |
| **卵黄嚢腫瘍** | 若年女性。AFP高値。 |
