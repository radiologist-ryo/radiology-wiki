## 概要

未熟奇形腫（immature teratoma）は卵巣胚細胞性腫瘍の悪性型であり、未熟な神経組織（神経管・神経外胚葉由来）を含む点が成熟奇形腫と異なる。成熟奇形腫に似た脂肪・石灰化成分を含みながら、神経性充実成分を伴う。悪性度はGrade 1〜3（神経組織の量で分類）で評価され、Grade 3は予後が比較的不良である。

---

## 疫学

20歳未満の若い女性に多く、卵巣悪性胚細胞腫瘍の中で2番目に多い。卵巣悪性腫瘍全体の1〜3%を占める。AFP（α-フェトプロテイン）・βHCGが上昇することがある。化学療法（BEP: bleomycin + etoposide + cisplatin）が有効であり、若年女性の妊孕性温存が考慮される。

---

## 病理

肉眼所見では大型の充実嚢胞混合性腫瘤で、脂肪・毛髪・石灰化と神経組織（脳組織様の軟部成分）が混在する。組織学的には未熟な神経管・神経芽細胞が認められ、その量によりGrade 1〜3に分類される。

---

## 画像所見

### 超音波（US）

超音波では大型の充実嚢胞混合性腫瘤として描出され、内部に高エコーの脂肪成分と低エコーの充実成分が混在する。以下に所見をまとめる。

| 所見 | 特徴 |
|------|------|
| 形態 | 大型充実嚢胞混合性 |
| 内部 | 不均一（脂肪・充実成分混在） |
| 石灰化 | 点状〜粗大石灰化 |

### CT

CTでは成熟奇形腫成分（脂肪の低吸収域・石灰化）に加えて、神経性充実成分（軟部組織密度）を認めることが特徴的である。充実成分は造影で増強を示す。以下に所見をまとめる。

| 所見 | 特徴 |
|------|------|
| 脂肪成分 | 低吸収域（成熟奇形腫成分） |
| 石灰化 | あり |
| 充実成分 | 神経性成分（造影増強） |
| 大きさ | 大型（10〜20 cm以上） |

### MRI

MRIでは脂肪によるT1高信号と神経性充実成分（T2中等度〜高信号）が共存する。以下に所見をまとめる。

| 所見 | 特徴 |
|------|------|
| T1強調像 | 脂肪成分高信号＋充実成分中等度信号 |
| T2強調像 | 充実成分：中等度〜高信号（神経組織） |
| DWI | 充実成分に拡散制限 |
| 造影 | 充実成分が増強 |

## 鑑別診断

| 疾患 | 鑑別のポイント |
|------|---------------|
| **成熟嚢胞性奇形腫** | 充実成分（神経組織）なし。良性。 |
| **卵黄嚢腫瘍** | AFP著明高値。充実性主体。脂肪なし。 |
| **高異型度漿液性癌（HGSC）** | 年長女性。腹膜播種主体。 |
| **混合型胚細胞腫瘍** | 複数の組織型が混在（未熟奇形腫＋卵黄嚢腫瘍など）。 |
