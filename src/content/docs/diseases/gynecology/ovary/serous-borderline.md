## 概要

漿液性境界悪性腫瘍（serous borderline tumor; SBT）は、良性と悪性の中間的な生物学的挙動を示す上皮性卵巣腫瘍であり、WHO分類では漿液性腫瘍の「境界悪性（低悪性度）」に分類される。組織学的には核異型と細胞増殖を示すが、間質浸潤を認めないことが定義上の条件である。腹膜播種（implant）を伴うことがあり、浸潤性implantを有する場合は予後が不良となる。

---

## 疫学

全年齢に発生するが、20〜40歳代の比較的若い女性に多く、卵巣悪性腫瘍の10〜15%を占める。両側性は約25〜30%にみられ、漿液性嚢胞腺腫よりも高率である。5年生存率は95%以上と予後は良好であるが、晩期再発があり長期フォローアップが必要とされる。

---

## 病理

肉眼所見では内腔に乳頭状突起（papillary projection）を伴う嚢胞性腫瘤として認められる。組織学的には核異型を示す漿液性上皮細胞が乳頭状に増殖するが、間質への浸潤はない。腹膜implantは非浸潤性と浸潤性に区別され、後者は予後不良因子となる。砂粒体（psammoma body）を高頻度に伴う。

---

## 画像所見

### 超音波（US）

超音波では嚢胞性腫瘤の内壁に乳頭状突起として描出される充実成分が特徴的である。血流シグナルを乳頭状突起内に認めることが多く、これは良性との鑑別に有用な所見とされる。壁の不整や微細な隔壁の増加も認められる。以下に所見をまとめる。

| 所見 | 特徴 |
|------|------|
| 乳頭状突起 | 嚢胞内壁に乳頭状充実成分（特徴的） |
| 壁・隔壁 | 不整、薄〜中等度の隔壁 |
| 血流 | 乳頭状突起内にカラードプラ陽性 |
| 腹水 | 少量の腹水を伴うことがある |

### CT

CTでは薄壁の嚢胞性腫瘤内に乳頭状の造影増強を示す充実成分を認める。腹膜播種は小結節状の腹膜肥厚として描出される。砂粒体による微細石灰化が腫瘍内や腹膜に散在することがある。以下に所見をまとめる。

| 所見 | 特徴 |
|------|------|
| 充実成分 | 乳頭状・造影増強（+++） |
| 嚢胞壁 | 薄〜中等度の壁肥厚 |
| 腹膜播種 | 小結節状腹膜肥厚（進行例） |
| 石灰化 | 砂粒体による微細石灰化 |

### MRI

MRIでは乳頭状突起がT2強調像で中等度信号の充実成分として描出され、造影後に増強される。T1強調像での腹膜播種検出にはDWIが有用である。以下に所見をまとめる。

| 所見 | 特徴 |
|------|------|
| T2強調像 | 嚢胞壁に中等度信号の乳頭状突起 |
| 造影 | 乳頭状突起が増強 |
| DWI | 乳頭状突起に軽度の拡散制限を示すことがある |

---

## 鑑別診断

乳頭状突起の有無が漿液性嚢胞腺腫との最大の鑑別点であるが、確定診断は病理組織学的検査による。

| 疾患 | 鑑別のポイント |
|------|---------------|
| **漿液性嚢胞腺腫** | 乳頭状突起なし。充実成分なし。 |
| **高異型度漿液性癌（HGSC）** | 大きな充実成分、腹水、腹膜播種が顕著。 |
| **粘液性境界悪性腫瘍** | 多房性。内容液が粘液性（高CT値）。 |
| **漿液性嚢胞腺癌** | 浸潤性間質増殖。大型充実成分。CA125著明高値。 |
