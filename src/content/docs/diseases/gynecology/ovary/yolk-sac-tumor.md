## 概要

卵黄嚢腫瘍（yolk sac tumor; YST）は内胚葉洞腫瘍（endodermal sinus tumor）とも呼ばれ、卵黄嚢に類似した構造を示す卵巣悪性胚細胞腫瘍である。悪性胚細胞腫瘍の20%を占め、血清AFP（α-フェトプロテイン）の著明高値が診断的価値を持つ。BEP療法への反応性は良好であり、早期診断・早期治療により予後改善が期待できる。

---

## 疫学

10〜20歳代の若い女性に多く発症し、乳児〜小児にもみられる。急速に増大する傾向があり、発見時に大型であることが多い。AFP著明高値（通常 > 1000 ng/mL）が特徴的であり、治療効果判定・再発評価に有用である。

---

## 病理

肉眼所見では大型の充実嚢胞混合性腫瘤で、出血・壊死を高頻度に伴う。組織学的にはSchiller-Duval小体（糸球体様構造）が特徴的で、AFP産生細胞が認められる。均一な細胞からなる充実型・嚢胞型・乳頭型など複数のパターンが存在する。

---

## 画像所見

### 超音波（US）

超音波では大型の充実嚢胞混合性腫瘤として描出される。充実成分への血流が豊富で、内部に出血・壊死による不均一な無エコー域を認める。以下に所見をまとめる。

| 所見 | 特徴 |
|------|------|
| 形態 | 大型充実嚢胞混合性 |
| 充実成分 | 不均一（出血・壊死） |
| 血流 | 充実成分に豊富 |
| 急速増大 | 急速増大の傾向 |

### CT

CTでは大型の充実嚢胞混合性腫瘤として描出され、充実成分は造影で増強を示す。出血・壊死による不均一な内部構造が特徴的である。以下に所見をまとめる。

| 所見 | 特徴 |
|------|------|
| 形態 | 大型充実嚢胞混合性 |
| 充実成分 | 造影増強（不均一） |
| 出血・壊死 | 高〜低吸収域の混在 |
| リンパ節 | 後腹膜リンパ節腫大を伴うことあり |

### MRI

MRIでは充実成分にDWI拡散制限を認め、出血をT1高信号として同定できる。以下に所見をまとめる。

| 所見 | 特徴 |
|------|------|
| T1強調像 | 出血部：高信号。充実成分：中等度 |
| T2強調像 | 充実成分：中等度〜高信号（不均一） |
| DWI | 充実成分に著明な拡散制限 |
| 造影 | 充実成分が不均一増強 |

## 鑑別診断

| 疾患 | 鑑別のポイント |
|------|---------------|
| **無性細胞腫** | AFP正常〜軽度上昇。LDH・PLAP上昇。均一充実性。 |
| **未熟奇形腫** | 脂肪成分あり。AFP軽度上昇のこともある。 |
| **高異型度漿液性癌（HGSC）** | 年長女性。CA125高値。腹膜播種。 |
| **顆粒膜細胞腫** | 内分泌症状。インヒビン高値。 |
