## 概要

喉頭は喉頭蓋上端から輪状軟骨下縁まで約 5 cm の範囲を占める。喉頭癌のうち声門癌が半数以上を占め、次いで声門上癌、声門下癌は稀である。声門癌は早期から嗄声を来すため早期発見されやすい。進行癌では喉頭全摘出術など侵襲性の高い手術が必要となる。

## 喉頭癌のTNM分類概略

UICC 第 9 版での下咽頭癌 T 分類変更はない<Cite id={1} />。T 分類は病変主在部位の亜部位（声門上・声門・声門下）によって異なり、内視鏡所見と照らし合わせた局在診断が重要である。画像評価は主に CT を用い、T3 以降の深部組織浸潤・リンパ節転移・遠隔転移の確認に意義がある。深部組織浸潤の評価では喉頭蓋前間隙・傍声帯間隙・甲状軟骨・輪状軟骨・喉頭外頸部軟部組織への進展を評価する<Cite id={2} /><Cite id={3} />。

### 声門上部T分類

| 分類 | 定義 |
|------|------|
| T1 | 1 亜部位に限局、声帯可動性正常 |
| T2 | 隣接する複数亜部位または声門上部外の粘膜に浸潤、声帯固定なし |
| T3 | 喉頭に限局、声帯固定あり、または傍声帯間隙浸潤、または甲状軟骨内側骨皮質浸潤 |
| T4a | 甲状軟骨（内・外骨皮質）貫通、または気管・食道・外舌筋・前頸筋群・甲状腺への浸潤 |
| T4b | 椎前筋膜浸潤、頸動脈全周性浸潤、縦隔進展 |

### 声門部T分類

| 分類 | 定義 |
|------|------|
| T1a | 1 側声帯に限局、声帯可動性正常 |
| T1b | 両側声帯に及ぶ（前連合含む）、声帯可動性正常 |
| T2 | 声門上または声門下への進展、または声帯運動障害 |
| T3 | 喉頭に限局、声帯固定あり、傍声帯間隙浸潤、または甲状軟骨内側骨皮質浸潤 |
| T4a | 声門上と同（輪状軟骨浸潤も含む） |
| T4b | 声門上と同 |

### 声門下部T分類

| 分類 | 定義 |
|------|------|
| T1 | 声門下に限局 |
| T2 | 声帯への進展（声帯可動性正常または障害） |
| T3 | 喉頭に限局、声帯固定あり |
| T4a | 輪状軟骨・甲状軟骨浸潤、または気管・食道・外舌筋・前頸筋群・甲状腺への浸潤 |
| T4b | 声門上と同 |

### N分類・M分類・病期分類

| 分類 | 定義 |
|------|------|
| N0 | 領域リンパ節転移なし |
| N1 | 同側単発、最大径 3 cm 以下、cENE なし |
| N2a | 同側単発、最大径 3〜6 cm、cENE なし |
| N2b | 同側多発、最大径 6 cm 以下、cENE なし |
| N2c | 両側/対側、最大径 6 cm 以下、cENE なし |
| N3a | 最大径 6 cm 超、cENE なし |
| N3b | cENE あり |
| M0 | 遠隔転移なし |
| M1 | 遠隔転移あり |

|------|---|---|---|

## 亜部位

- **声門上部**: ①舌骨上喉頭蓋、②披裂喉頭蓋ヒダ（喉頭面）、③披裂、④舌骨下喉頭蓋、⑤仮声帯
- **声門部**: ①声帯、②前連合、③後連合
- **声門下部**: 単独、気管との境界は輪状軟骨下縁

## 局所進展（T分類）

### T1（早期癌）

声帯腫瘍は初期から嗄声を来すため早期発見が多い。声門癌 T1 ではリンパ節転移リスクが非常に低く、経口的切除術や喉頭限局範囲の放射線治療単独が適応となる。CT は安静呼吸下での撮影が重要であり、呼吸停止で声帯内転・声門閉鎖が生じると早期表在癌の評価が困難になる。声門上癌・声門下癌の T1 は各 1 亜部位に限局する。

### T2（声門下進展）

声門下癌・声門癌 T2 の声門下進展は内視鏡での評価が困難なことが多く、CT による確認が重要である。前方優位の声門下進展は喉頭前リンパ節へのリンパ路となるため、リンパ節転移に注意が必要である<Cite id={4} />。

### T3（喉頭蓋前間隙・傍声帯間隙・甲状軟骨内側骨皮質浸潤）

深部組織浸潤（喉頭蓋前間隙・傍声帯間隙）または甲状軟骨内側皮質浸潤が T3 の要件となる。喉頭蓋前間隙・傍声帯間隙は脂肪を含む結合組織であり、通常 CT で低吸収を呈する。浸潤ではやや高吸収となり、左右差や造影効果で判断する。

甲状軟骨浸潤については、軟骨の不整侵食・骨融解などの破壊性変化が浸潤を示唆する。横断像のみでは立体構造と病変境界が捉えにくい場合があり、矢状断・冠状断を含む 3 方向評価が有用である。腫瘍浸潤による硬化性変化は正常骨化軟骨・二次性炎症性骨硬化と区別が困難であり、CT での過剰評価に注意が必要である。CT で甲状軟骨全体が腫瘍と同様の軟部濃度を呈する場合、軟骨浸潤の診断が非常に困難となる<Cite id={5} />。

MRI は CT より甲状軟骨浸潤の感度が高く、軟骨内に腫瘍と同様の低信号（T1WI）・やや高信号（T2WI）・造影効果を認める場合に浸潤を示唆する。ただし腫瘍に伴う炎症性変化（直接浸潤ではない）も同様の信号変化を来すため特異度がやや低い<Cite id={6} />。Dual-energy CT のヨードオーバーレイ像では浸潤部位のヨード造影が明瞭化し、従来 CT・MRI より診断能が向上する<Cite id={6} />。

### T4（喉頭外組織浸潤）

**T4a**: 甲状軟骨を越えた喉頭外組織（気管・舌深層筋/外舌筋を含む頸部軟部組織・前頸筋群・甲状腺・食道）への浸潤。

甲状軟骨貫通以外の進展経路として、甲状軟骨後縁から回り込む経路、甲状舌骨膜を介する経路、輪状甲状膜を介する経路がある<Cite id={6} /><Cite id={7} />。また解剖学的変異として甲状孔（甲状軟骨後上部の孔）から進展する経路もある。

甲状軟骨後縁を包み込む wrap around spread は下咽頭癌の梨状陥凹からの喉頭外進展経路としても重要であり、咽頭収縮筋の付着部は甲状軟骨後端外側に位置する。甲状舌骨膜は舌骨と甲状軟骨間の線維性膜であり、後外側部を上喉頭動脈と上喉頭神経内枝からなる上喉頭神経血管束が貫通する。甲状舌骨膜を介する進展は上喉頭神経血管束に沿うことが多く、同血管束はリンパ路としても機能し内深頸領域へのリンパ節転移を伴う。

甲状軟骨と輪状軟骨は後方では輪状甲状関節、前方では輪状甲状靱帯によって連結され、その間に輪状甲状膜が存在する。輪状甲状膜は主に上甲状腺動脈の枝が貫通し、神経血管束を介した進展経路となりやすい。いずれの経路においても喉頭外組織進展を示す境界面の不整や周囲脈管の巻き込みを丁寧に拾うことが重要である。甲状軟骨浸潤・声帯固定（T3）を伴わない T4a では喉頭機能温存治療が選択肢となる<Cite id={7} />。声門下部 T4a には輪状軟骨浸潤が含まれる。

**T4b**: 椎前間隙浸潤・頸動脈全周性浸潤・縦隔進展。椎前筋膜は比較的強固であり椎前間隙への進展頻度は低く、画像で椎前間隙の脂肪を確認することで除外できる<Cite id={3} />。

## リンパ節転移（N分類）

短径 10 mm 以上・辺縁不整・内部不均一・球形化・左右差・リンパ節門消失などの CT 所見と、超音波・PET-CT での FDG 集積を総合して判断する<Cite id={8} />。**喉頭前リンパ節**（特に輪状甲状膜前方 = Delphian リンパ節）は通常腫大しないため、画像で確認可能なサイズであれば喉頭癌や甲状腺癌からの転移の可能性が高い。

## 遠隔転移（M分類）

M1 では肺転移が最多であり、次いで骨転移・肝転移が続く。遠隔転移患者の予後因子は T 分類・脳転移・外科治療の有無である<Cite id={9} />。喉頭癌を含む頭頸部癌は重複癌の発生頻度が高く、頭頸部・肺・食道で多いため、上部消化管内視鏡を含む定期的な経過観察が必要である<Cite id={3} />。

## まとめ

進行癌に対しても喉頭温存治療（化学放射線療法など）が選択肢となってきているが、甲状軟骨浸潤・輪状披裂関節浸潤症例では局所制御率が不良であり長期喉頭機能低下の予測因子となる<Cite id={7} />。臨床病期決定にとどまらず、機能温存を含めた治療戦略のための詳細な読影が求められる。
