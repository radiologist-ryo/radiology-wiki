## 概要

鼻副鼻腔領域の悪性腫瘍は様々な組織型があるが、扁平上皮癌が最多である。TNM分類は AJCC/UICC および頭頸部癌取扱い規約第6版補訂版に準拠して行われる<Cite id={1} />。2025年に UICC 第9版へ改訂されたが<Cite id={2} />、鼻副鼻腔癌の主要項目に大きな変更はない。上顎洞癌と鼻腔/篩骨洞癌ではT因子の一部が異なり、N・M因子は共通である。

治療方針としては、I/II期は外科的切除単独、III/IV期は外科切除と術後放射線療法の組み合わせが基本となる。T3以上ではIII期以上が確定し、局所制御率の低下・予後不良と関連するため、画像診断ではT3以上を規定する進展範囲の正確な評価が不可欠である。

## 上顎洞癌のT分類

| 分類 | 定義 |
|------|------|
| T1 | 上顎洞粘膜に限局、骨びらん・骨破壊なし |
| T2 | 骨びらん・骨破壊を伴うが、後壁・翼突窩・眼窩内容への進展なし |
| T3 | 皮下組織・上顎洞後壁・眼窩底/眼窩内側壁・翼突窩・篩骨洞への進展 |
| T4a | 眼窩前部・頰部皮膚・翼状突起・側頭下窩・篩板・蝶形骨洞・前頭洞への進展 |
| T4b | 眼窩先端・硬膜・脳・中頭蓋窩・V2以外の脳神経・鼻咽腔・斜台への進展 |

## 前方進展(上顎洞癌)

### 皮下組織(T3)

顔面軟部組織は表層から深部へ①表皮、②浅脂肪層、③SMAS（superficial musculoaponeurotic system）、④浅顔面筋、⑤深筋膜＋深脂肪層、⑥保持靱帯の6層構造で構成される<Cite id={3} />。SMASは上顎洞前壁に沿って線状の軟部吸収値として描出される。T分類上の「皮下組織」は浅脂肪層・SMAS・浅顔面筋・深筋膜＋深脂肪層・保持靱帯を含む。腫瘍が上顎洞前壁を越えて皮下組織へ突出、あるいは隣接脂肪織の混濁を認める場合に皮下組織浸潤と判定する。

### 頬部皮膚(T4a)

皮膚は表皮と真皮からなり、CT・MRI での区別は困難である。皮膚の肥厚・断裂・異常造影効果を認めた場合に皮膚浸潤と判定する。

## 後方進展(上顎洞癌)

### 上顎洞後壁(T3)

上顎洞壁への腫瘍浸潤形式は、破壊性・浸透性・膨張性の3パターンに分類される。上顎洞扁平上皮癌では破壊性が83%、浸透性が10%、膨張性が7%を占める<Cite id={4} />。

### 翼突窩(T3)

翼突窩は蝶形骨翼状突起の内側板と外側板の間に位置し、内側翼突筋・口蓋帆張筋を含む解剖学的空間である。上顎洞後壁の骨破壊に続いて腫瘍が翼突窩脂肪織を置換する形で進展する。

### 翼状突起(T4a)

翼状突起は蝶形骨体・大翼から下方に突出し、外側板と内側板で構成される。腫瘍接触部位の骨破壊・骨吸収・高度骨硬化をもって翼状突起浸潤と判定する。MRI では炎症や浮腫による異常信号も生じうるため、腫瘍類似の異常信号や異常造影効果の有無を確認して判定する。

### 側頭下窩(T4a)

側頭下窩は前方を上顎骨後面、後方を側頭骨乳突部、外側を下顎枝・頰骨弓、内側を翼状突起外側板で囲まれる空間であり、固有の筋膜被膜をもたない。咀嚼筋間隙・傍咽頭間隙・頰間隙の一部が組み合わさった領域で、側頭筋下部・内外翼突筋・顎動脈・咽頭（翼突）静脈叢・V3を含むが、咬筋は含まれない。

## 眼窩内進展(上顎洞癌)

上顎洞癌の60〜80%に眼窩内進展を認め、予後不良因子である。眼窩底/眼窩内側壁への進展はT3、眼窩内容前部はT4a、眼窩先端はT4bに分類される。外眼筋・眼窩尖部・眼球結膜・強膜への浸潤では眼球摘出を選択することが多く、眼窩骨膜を超えた進展では眼窩内容温存が困難となる。

### 眼窩底/眼窩内側壁(T3)

CT で眼窩骨壁の骨破壊・骨吸収を認めた場合に進展と判定する。眼窩骨膜は MRI で線状低信号として描出され<Cite id={5} />、骨壁との区別は困難なため CT との相補的評価が重要である。CT で眼窩骨壁破壊を認め、かつ MRI で眼窩骨膜の連続性が保たれていればT3と判定する。

### 眼窩内容前部(T4a)

眼窩内容前部は眼窩脂肪織・外眼筋・眼球・涙腺・神経・血管を含む。CT で眼窩骨壁破壊、MRI で眼窩骨膜断裂と腫瘍の眼窩脂肪織への突出、あるいは隣接眼窩脂肪織の混濁を認めた場合に脂肪織浸潤と判定する。外眼筋の腫大・異常信号・異常造影効果は外眼筋浸潤の根拠となるが感度は低く、外眼筋偏位は感度が高いが偽陽性が多い<Cite id={6} />。

### 眼窩先端(T4b)

眼窩先端は視神経管（第II脳神経通過）と上眼窩裂（第III・IV・VI・V1通過）を含む。進展経路としては、翼口蓋窩→下眼窩裂→眼窩先端、または眼窩壁破壊→眼窩内直接浸潤がある。視神経管の骨壁破壊で視神経管浸潤と判定し、上眼窩裂の脂肪消失や異常造影効果で上眼窩裂浸潤と判定する。

## 頭蓋内進展(上顎洞癌)

硬膜浸潤・脳浸潤・V2以外の脳神経浸潤はいずれも予後不良でT4bに分類される。硬膜浸潤までは外科的切除の適応となりうるが、海綿静脈洞・内頸動脈・脳実質浸潤では手術適応外となることが多い。進展経路としては、篩骨洞→篩板→前頭蓋窩（篩板浸潤はT4a）と、側頭下窩→中頭蓋窩（中頭蓋窩浸潤はT4b）の2経路がある。

### 硬膜(T4b)

硬膜の肥厚が2mm以上<Cite id={7} /><Cite id={8} />または5mm以上<Cite id={9} />であれば浸潤を示唆する。硬膜の結節状造影効果は複数の報告でいずれも信頼性が高い所見とされる<Cite id={7} /><Cite id={8} /><Cite id={9} />。一方、硬膜の線状造影効果は鼻副鼻腔/頭蓋底腫瘍でしばしば認められるが、感染・炎症・線維化・血管増生といった反応性変化でも生じうるため、腫瘍浸潤としての特異性は高くない<Cite id={9} /><Cite id={10} /><Cite id={11} />。

### 脳(T4b)

脳実質の異常造影効果は信頼性のある浸潤所見である。脳浮腫は腫瘍浸潤がなくても生じうるため、造影効果のない浮腫のみでは浸潤を断定できない場合がある<Cite id={8} /><Cite id={12} />。

### V2以外の脳神経(T4b)

翼口蓋窩経由のV2浸潤は高頻度であるが、V2の浸潤はT4には含まれない（V2「以外」の脳神経浸潤でT4b）。上眼窩裂・視神経管への進展は脳神経浸潤を示唆し、側頭下窩への進展ではV3浸潤も生じうる。

## 鼻腔癌・篩骨洞癌のT分類

| 分類 | 定義 |
|------|------|
| T1 | 鼻腔または篩骨洞の一亜部位に限局、骨壁びらん・骨破壊なし |
| T2 | 鼻腔/篩骨洞の2亜部位以上への進展、または隣接鼻副鼻腔への進展 |
| T3 | 眼窩内側壁・眼窩底・上顎洞・口蓋・篩板への進展 |
| T4a | 眼窩前部・鼻あるいは頰部皮膚・頭蓋内最小限の進展・篩板越え前頭蓋窩への進展 |
| T4b | 眼窩先端・硬膜・脳・中頭蓋窩・鼻咽腔・斜台・V2以外の脳神経への進展 |

## 前頭蓋底進展(鼻腔/篩骨洞癌)

### 篩板(T3)

鼻腔/篩骨洞癌では篩板浸潤がT3に相当するが、上顎洞癌ではT4aとなる点に注意する。篩板には嗅神経が通過する篩骨孔が多数存在し、扁平上皮癌浸潤により骨吸収・骨欠損が生じる。

### 前頭蓋窩(最小限の進展)(T4a)

篩板破壊＋前頭蓋窩への腫瘍突出を認めても、硬膜の結節状造影効果を伴わない場合は最小限の前頭蓋窩浸潤と判定するのが妥当である。線状造影効果のみの場合も同様に最小限の浸潤として扱うことができる。

## N分類・M分類

頸部リンパ節転移(N)・遠隔転移(M)の定義はUICCに準拠する<Cite id={2} />。上顎洞扁平上皮癌の頸部リンパ節転移頻度は他部位の頭頸部扁平上皮癌より低く3〜20%にとどまる。遠隔転移は初診時5%未満であり、肺・骨・肝への転移が多い。

| 分類 | 定義 |
|------|------|
| N0 | 領域リンパ節転移なし |
| N1 | 同側単発、最大径3cm以下、節外浸潤なし |
| N2a | 同側単発、最大径3〜6cm、節外浸潤なし |
| N2b | 同側多発、最大径6cm以下、節外浸潤なし |
| N2c | 両側または対側、最大径6cm以下、節外浸潤なし |
| N3a | 最大径6cm超、節外浸潤なし |
| N3b | 節外浸潤あり |
| M0 | 遠隔転移なし |
| M1 | 遠隔転移あり |

## まとめ

T3以上を規定する評価項目の正確な把握が治療方針決定に不可欠である。上顎洞癌に特有の評価項目は皮下組織・上顎洞後壁・翼突窩・側頭下窩、鼻腔/篩骨洞癌に特有の評価項目は口蓋・前頭蓋窩である。眼窩内容前部や硬膜への進展は判定が困難な場合があり、CT と MRI の相補的評価と各所見の信頼性を考慮した慎重な評価が求められる。

## TNM分類

鼻副鼻腔癌の T分類は上顎洞と鼻腔・篩骨洞で分けて規定される。前頭洞・蝶形骨洞原発については規約・UICC ともに規定がない。

### 上顎洞 — T分類

| 分類 | UICC第9版(2025年) | 頭頸部癌取扱い規約 第6版補訂版 |
|---|---|---|
| T1 | 上顎洞粘膜に限局、骨びらん・骨破壊なし | 上顎洞粘膜に限局し、骨吸収または骨破壊を認めない |
| T2 | 骨びらん・骨破壊を伴うが、後壁・翼突窩・眼窩内容への進展なし | 骨吸収や骨破壊のある腫瘍、硬口蓋や中鼻道に進展する腫瘍を含むが、上顎洞後壁および翼状突起に進展する腫瘍を除く |
| T3 | 皮下組織・上顎洞後壁・眼窩底/眼窩内側壁・翼突窩・篩骨洞への進展 | 上顎洞後壁の骨、皮下組織、眼窩底または眼窩内側壁、翼突窩、篩骨洞のいずれかに浸潤する |
| T4a | 眼窩前部・頰部皮膚・翼状突起・側頭下窩・篩板・蝶形骨洞・前頭洞への進展 | 眼窩内容前部、頬部皮膚、翼状突起、側頭下窩、篩板、蝶形骨洞、前頭洞のいずれかに浸潤する |
| T4b | 眼窩先端・硬膜・脳・中頭蓋窩・V2以外の脳神経・鼻咽腔・斜台への進展 | 眼窩尖端、硬膜、脳、中頭蓋窩、三叉神経第二枝(V2)以外の脳神経、上咽頭、斜台のいずれかに浸潤する |

### 鼻腔・篩骨洞 — T分類

| 分類 | UICC第9版(2025年) | 頭頸部癌取扱い規約 第6版補訂版 |

| T1 | 鼻腔または篩骨洞の一亜部位に限局、骨壁びらん・骨破壊なし | 骨浸潤の有無に関係なく、鼻腔または篩骨洞の1亜部位に限局する |

| T3 | 眼窩内側壁・眼窩底・上顎洞・口蓋・篩板への進展 | 眼窩内側壁または眼窩底、上顎洞、口蓋、篩板のいずれかに浸潤する |
| T4a | 眼窩前部・鼻あるいは頰部皮膚・頭蓋内最小限の進展・篩板越え前頭蓋窩への進展 | 眼窩内容前部、外鼻の皮膚、頬部皮膚、前頭蓋窩(軽度進展)、翼状突起、蝶形骨洞、前頭洞のいずれかに浸潤する |
| T4b | 眼窩先端・硬膜・脳・中頭蓋窩・鼻咽腔・斜台・V2以外の脳神経への進展 | 眼窩尖端、硬膜、脳、中頭蓋窩、三叉神経第二枝(V2)以外の脳神経、上咽頭、斜台のいずれかに浸潤する |

| 分類 | 定義 |
|---|---|
| N0 | 領域リンパ節転移なし |
| N1 | 同側の単発性リンパ節転移で最大径が3cm以下かつ節外浸潤なし |
| N2a | 同側の単発性リンパ節転移で最大径が3cmをこえるが6cm以下かつ節外浸潤なし |
| N2b | 同側の多発性リンパ節転移で最大径が6cm以下かつ節外浸潤なし |
| N2c | 両側または対側のリンパ節転移で最大径が6cm以下かつ節外浸潤なし |
| N3a | 最大径が6cmをこえるリンパ節転移で節外浸潤なし |
| N3b | 単発性または多発性リンパ節転移で臨床的節外浸潤あり |

### N分類の補足

鼻副鼻腔癌のリンパ節転移は他の頭頸部癌と比較して頻度は低いが、進行癌(T4a/T4b)や扁平上皮癌などでは転移リスクがある。神経周囲進展の評価が重要であり、上顎洞癌では三叉神経第二枝(V2)に沿った翼口蓋窩から正円孔への進展、眼窩下神経(眼窩下孔)に沿った進展がある。鼻腔・篩骨洞癌では蝶口蓋孔から翼口蓋窩、正円孔への進展を確認する。眼窩進展、頭蓋内進展(硬膜の2mm以上の肥厚、結節状肥厚、T2強調像の低信号の断裂が硬膜浸潤を示唆)、神経周囲進展は T4a/T4b 分類に直結するため画像評価の重点となる。
