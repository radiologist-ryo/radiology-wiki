## 概要

甲状腺病変の良悪性を含む質的診断は超音波・超音波ガイド下生検が第一選択である。嗄声・喘鳴・喀血など臨床症状があれば悪性を疑い、CT/MRIによる局所浸潤・遠隔転移評価が行われる。正確な病期診断・手術適応を含む治療方針決定に重要な役割を担う。

## 国内外の甲状腺癌病期診断の特徴

### UICC第9版における甲状腺癌TNM分類

UICC第9版<Cite id={1} />のT分類を以下に示す。

| T分類 | 定義 |
|-------|------|
| cT1a | ≤1cm、甲状腺内限局 |
| cT1b | 1-2cm、甲状腺内限局 |
| cT2 | 2-4cm、甲状腺内限局 |
| cT3a | &gt;4cm、甲状腺内限局 |
| cT3b | サイズによらず、前頸筋群(胸骨舌骨筋・胸骨甲状筋・肩甲舌骨筋)または副甲状腺への浸潤(UICC第9版で副甲状腺追加) |
| cT4a | 皮下脂肪組織・喉頭・気管・食道・反回神経・胸鎖乳突筋浸潤 |
| cT4b | 椎前筋膜・縦隔内血管浸潤、頸動脈の全周性取り囲み |

NM分類はUICC第8版と同様であり、HPV関連中咽頭癌や唾液腺癌の第9版で追加されたiENEは甲状腺癌では導入されていない。

### UICC第9版における甲状腺癌病期診断

主な病理組織型は乳頭癌・濾胞癌・髄様癌・未分化癌の4種類で、病理型によって予後が異なる。予後良好な乳頭癌と濾胞癌は同一の病期設定が用いられる。

年齢を境とした判定が特徴であり、55歳未満では遠隔転移なしでstage I、遠隔転移ありでもstage IIとなる。55歳以上ではTNMに応じた詳細な病期区分が適用される。未分化癌は予後不良であり、いずれのT分類でもstage IVとなる。

### 甲状腺未分化癌(注意点)

画像所見から病理組織学的分類は困難なことが多いが、未分化癌の存在には注意が必要である。急速増大・予後不良であり、乳頭癌/濾胞癌など先行する分化癌が未分化転化して発症する例がある。画像的特徴として、先行病変の影響と考えられる卵殻状の石灰化所見、出血・内部壊死・周囲への浸潤性が報告されている<Cite id={2} />。

### 甲状腺癌取扱い規約 第9版のポイント

わが国では2023年10月より第9版が使用されている<Cite id={3} />。T分類はUICC第8版と同一である。

甲状腺腫瘍の肉眼的腺外浸潤所見(術中所見、surgical findings)としてsEx分類があることが特徴である<Cite id={4} />。

| sEx分類 | 浸潤範囲 |
|---------|---------|
| sEx0 | 腺外浸潤なし |
| sEx1a | 甲状腺周囲脂肪組織 |
| sEx1b | 前頸筋群・副甲状腺 |
| sEx2a | 気管外膜/軟骨・反回神経・食道筋層・輪状甲状筋・下咽頭収縮筋 |
| sEx2b | 皮下脂肪・気管/食道粘膜・内頸/腕頭静脈・喉頭/咽頭内・胸鎖乳突筋 |
| sEx3 | 椎骨前筋膜・縦隔大血管・頸動脈 |

このsEx分類はわが国規約特有の分類であり、浸潤臓器だけでなく深達度を判定するものである。実際の分類は手術所見でなされるため読影レポートへの記載は不要であるが、浸潤の深達度が予後因子として重要である<Cite id={4} />。

## CT・MRIにおける甲状腺癌診断のポイント

大部分を占める高分化甲状腺癌の予後は極めて良好であるが、高分化甲状腺癌で甲状腺外浸潤を示す症例は約13%存在し<Cite id={5} />、稀ではない。甲状腺内限局の10年生存率91%に対し、甲状腺外浸潤を伴うと45%まで低下し<Cite id={6} />、予後に直結する。浸潤部位の頻度は反回神経47%、気管37%、大血管など30%、食道21%と報告されている<Cite id={7} />。

### 反回神経浸潤(T4a)

嗄声・嚥下障害・気道狭窄の原因となり、QOLに大きく関係する。術前から声帯麻痺がある症例は高度浸潤が多く神経合併切除が推奨される。術前に声帯麻痺がない場合でも術中所見で反回神経浸潤があれば、術中神経モニタリングの筋電図電位などを参考に術式を決定する。神経表層のみの浸潤であれば声帯機能温存を目的にシェービングが推奨される<Cite id={8} />。

**反回神経の解剖**:
- 迷走神経より分岐
- 右側: 鎖骨下動脈を前方から後方へ反回
- 左側: 大動脈弓下を前方から後方へ反回
- 食道枝・気管枝・下心臓枝を出し、気管食道溝(気管と食道の間)内を頭側へ上行、最終的に喉頭へ至る

**画像所見**:

反回神経自体のCT/MRI同定は困難であり、気管食道溝の変化や嗄声に伴う咽喉頭形態変化を見逃さないことが重要である。

- **直接的所見**: 腺外後方組織浸潤に伴う気管食道溝脂肪組織消失<Cite id={9} /><Cite id={10} />
- **診断精度**: MRIで1断面以上の完全な気管食道溝脂肪消失があれば反回神経浸潤の診断精度88%(感度94%/特異度82%)<Cite id={10} />
- **間接所見**: 患側声帯の傍正中偏位、喉頭室/梨状陥凹の拡大、披裂喉頭蓋ヒダの腫大、慢性期では患側声帯の萎縮・脂肪浸潤

**非反回喉頭神経(注意)**:

右鎖骨下動脈の起始異常がある場合、反回せず喉頭神経が輪状軟骨付近で迷走神経から直接喉頭に分布する。右側0.3-1.6%、左側0.04%と稀である<Cite id={11} /><Cite id={12} />。甲状腺手術での神経損傷防止のため反回神経走行の把握が重要であり、術前CTでの右鎖骨下動脈起始異常の指摘が望ましい。

### 気管浸潤(T4a)

喀血・呼吸困難を伴う出血/気道閉塞の可能性があり、死亡率と関連する<Cite id={11} />。気管は呼吸だけでなく発声・嚥下にも関わる重要臓器であり、生命予後とQOLを総合的に判断する。表層のみの浸潤であれば浸潤部中心に気道表層を鋭的切除し環状構造を温存するシェービングが行われるが、深層浸潤であれば気道合併切除が必要となる<Cite id={8} />。

**気管解剖**:
- 馬蹄形の構造
- 気管軟骨: MRIのT2強調像・T1強調像ともに低信号
- 気管粘膜および粘膜下組織: T2強調像で高信号、T1強調像で低信号

**高分化甲状腺癌気管浸潤の進行**:

表層から深層、内腔の順に進行する。ShinらはAnatomy based classificationとして軟骨浸潤と管腔内浸潤の有無により病理組織学的所見を分類した<Cite id={13} />。MahajanらはShin分類を基にしたCT grade分類を考案している<Cite id={14} />:

- Grade 0: 腫瘍と気管軟骨の距離が5mmを超える
- Grade 1: 外軟骨膜への接触
- Grade 2: 軟骨破壊の有無を問わず気管軟骨への浸潤
- Grade 3: 気管粘膜まで及ぶが粘膜隆起なし
- Grade 4: 気管粘膜の隆起・膨隆を伴う全層浸潤

**CT気管浸潤所見**: 腫瘍が気管に180°以上接する、腫瘍レベルでの気管内腔変形、腫瘍に隣接する気管粘膜の不整/肥厚/膨隆<Cite id={9} />

**MRI気管浸潤所見**: 腫瘍が気管周囲に180°以上接する、気管内腔への腫瘍存在、気管軟骨の信号異常<Cite id={15} />

気管軟骨信号異常は気管浸潤を反映するが、病理組織学的に軟骨間膜や外膜への浸潤を反映している場合があり、気管軟骨浸潤を一概に反映しているわけではないことに留意する。腫瘍が気管周囲に180°以上接する場合は少なくとも気管外膜への浸潤を示唆し、気管内腔への腫瘍存在は気管深層浸潤に該当するため気管合併切除を考慮する重要画像所見である<Cite id={15} />。

### 食道浸潤(T4a)

正常食道壁の病理組織学的構造は、筋層+外膜=外層、粘膜+粘膜下組織=内層から成る。MRIのT2強調像では内層が高信号、外層が低信号に描出される。食道浸潤の外科治療は筋層までであれば筋層切除、内腔まで及べば食道合併切除+再建となる<Cite id={8} />。

MRIは組織コントラスト分解能が高く食道壁層構造を同定しやすい。外層浸潤評価の感度82%/特異度94%が報告されており<Cite id={16} />、特に腫瘍浸潤による食道壁外層のT2強調像での低信号域断裂、造影画像での異常造影効果の進展で食道壁浸潤を考える。

CTは組織コントラスト分解能が乏しく食道壁多層構造の同定が困難な場合が多い。腫瘍が食道周囲に180°以上接する/正常食道壁構造の不明瞭化/内腔への浸潤を来している症例で食道浸潤を疑う<Cite id={9} />。

### 頸動脈浸潤(T4b)

頸動脈3層構造のうち中膜まで病変が及ぶと動脈再建が必要となり、頸動脈浸潤疑いでは血行再建術を考慮した手術準備が行われる<Cite id={8} />。

接触角度による評価:
- CTにおける甲状腺癌腺外浸潤評価では総頸動脈/内頸動脈との接触角度が180°以上で「浸潤あり」<Cite id={9} />
- MRIでの甲状腺分化癌頸動脈浸潤評価では接触角度225°基準で精度90%<Cite id={17} />
- 頭頸部腫瘍を含むMRIでの頸動脈浸潤評価では接触角度270°基準で感度100%/特異度88%/精度91%<Cite id={18} />

議論の余地はあるが、一般的に180°以下では浸潤なく剥離可能、270°以上で浸潤の可能性が高いと考えられている。接触角度によらず、画像的に頸動脈壁不明瞭化がある場合は病理学的浸潤ありの報告があり<Cite id={19} />、頸動脈壁に腫瘍と同様の造影効果を示す軟部組織置換があれば注意を要する。

## まとめ

甲状腺癌のUICC第9版TNM分類の変更点(cT3bへの副甲状腺追加)とCT/MRIにおける読影ポイントを概説した。甲状腺外浸潤評価ではCT/MRIが超音波以上の情報を提供することが多く、特に反回神経・気管・食道など周囲重要構造の評価は外科治療方針に寄与するため注意深い読影が重要である。

## TNM分類

甲状腺癌のT分類は UICC第9版と頭頸部癌取扱い規約 第6版補訂版で同一のため、UICC第9版のみを記載する。組織型(分化型・未分化型・髄様癌)で病期分類が異なる点に留意する。

### T分類(UICC第9版 / 頭頸部癌取扱い規約 第6版補訂版、共通)

| 分類 | 定義 |
|---|---|
| T1a | 最大径1cm以下、甲状腺内限局 |
| T1b | 最大径1cmをこえるが2cm以下、甲状腺内限局 |
| T2 | 最大径2cmをこえるが4cm以下、甲状腺内限局 |
| T3a | 最大径4cmをこえる、甲状腺内限局 |
| T3b | サイズによらず、前頸筋群(胸骨舌骨筋・胸骨甲状筋・肩甲舌骨筋)または副甲状腺への浸潤(UICC第9版で副甲状腺追加) |
| T4a | 皮下脂肪組織・喉頭・気管・食道・反回神経・胸鎖乳突筋浸潤 |
| T4b | 椎前筋膜・縦隔内血管浸潤、頸動脈の全周性取り囲み |

### N分類(UICC第9版 / 頭頸部癌取扱い規約 第6版補訂版、共通)

| 分類 | 定義 |
|---|---|
| N0 | 領域リンパ節転移なし |
| N1a | VI区またはVII区リンパ節転移(気管前・気管傍・喉頭前/Delphianリンパ節・上縦隔リンパ節) |
| N1b | 他の同側・両側・対側頸部(I〜V区)または咽頭後リンパ節転移 |

### M分類

| 分類 | 定義 |
|---|---|
| M0 | 遠隔転移なし |
| M1 | 遠隔転移あり |

### 病期分類

分化型甲状腺癌(乳頭癌・濾胞癌・低分化癌) — 55歳未満

| 病期 | T | N | M |
|---|---|---|---|
| I期 | Tに関係なく | Nに関係なく | M0 |
| II期 | Tに関係なく | Nに関係なく | M1 |

| 病期 | T | N | M |
|---|---|---|---|
| I期 | T1, T2 | N0 | M0 |
| II期 | T2, T3a, T3b | N0 | M0 |
| III期 | T1, T2, T3a, T3b | N1a | M0 |
| IVA期 | T4a / T1, T2, T3a, T3b | Nに関係なく / N1b | M0 |
| IVB期 | T4b | Nに関係なく | M0 |
| IVC期 | Tに関係なく | Nに関係なく | M1 |

髄様癌

| 病期 | T | N | M |
|---|---|---|---|
| I期 | T1 | N0 | M0 |
| II期 | T2, T3 | N0 | M0 |
| III期 | T1, T2, T3 | N1a | M0 |
| IVA期 | T4a / T1, T2, T3 | Nに関係なく / N1b | M0 |
| IVB期 | T4b | Nに関係なく | M0 |
| IVC期 | Tに関係なく | Nに関係なく | M1 |

|---|---|---|---|
