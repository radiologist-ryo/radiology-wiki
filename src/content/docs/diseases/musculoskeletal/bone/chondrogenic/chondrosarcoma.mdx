## 概要

軟骨肉腫（chondrosarcoma）は軟骨基質を産生する悪性骨腫瘍であり、原発性悪性骨腫瘍の中で骨肉腫に次いで頻度が高い（約25%）。

- **好発年齢**：40〜60歳代（中高年）
- **好発部位**：骨盤・近位大腿骨・近位上腕骨・肋骨・肩甲骨
- **悪性度分類**：Grade 1（低悪性度）〜 Grade 3（高悪性度）に分類。Grade 1は軟骨腫との境界が問題となる

発生様式により**中心性**（骨髄腔内発生）・**末梢性**（骨軟骨腫から二次性に発生）・**骨膜性**（骨表面）・**脱分化型**（dedifferentiated）に分類される。

## 画像所見

### 単純X線写真

軟骨基質による特徴的な石灰化パターンが診断の鍵となる。<Cite id={2} />

| 所見 | 特徴 |
|------|------|
| 基質石灰化 | コーン状・環状（rings-and-arcs）または半環状石灰化（popcorn calcification） |
| 辺縁 | Grade 1では比較的明瞭（IB型）、Grade 2〜3では不明瞭 |
| Endosteal scalloping | 内骨皮質の扇状侵食（軟骨肉腫では深くなる） |
| 骨皮質 | 菲薄化・膨化（Grade 1）〜破壊（Grade 2〜3） |
| 骨膜反応 | Grade 1では乏しい。高悪性度では骨膜反応あり |

### CT

CTは軟骨基質（rings-and-arcs pattern）の同定と内骨皮質侵食（endosteal scalloping）の定量評価に優れる。<Cite id={1} />

**Endosteal scallopingの評価**は軟骨腫と軟骨肉腫の鑑別に重要な指標である：

- 深さが骨皮質厚の2/3未満：軟骨腫を示唆
- 深さが骨皮質厚の2/3以上：軟骨肉腫を示唆

また、扇状侵食の長さが6.5 cmを超える場合や、骨皮質を貫通する場合は軟骨肉腫に特徴的である。

### MRI

MRIは腫瘍の進展範囲・軟部浸潤・関節内進展の評価に不可欠である。

| シーケンス | 所見 |
|-----------|------|
| T1強調像 | 均一な低〜中信号（硝子軟骨を反映） |
| T2強調像 | **非常に高信号**（硝子軟骨の高含水率を反映）。石灰化部は低信号の小結節として散在 |
| 造影T1 | 軟骨小葉間の線維血管性隔壁が増強（septal/peripheral enhancement）。悪性度が高いほど増強が中心部に及ぶ |
| DWI | Grade 1では拡散制限が乏しい。高悪性度では拡散制限を呈することがある |

## 軟骨腫との鑑別

長管骨に発生する軟骨腫（内軟骨腫）と低悪性度軟骨肉腫（Grade 1）の鑑別は画像・病理ともに最も困難な問題の一つである。<Cite id={1} /><Cite id={3} />

| 鑑別点 | 軟骨腫（内軟骨腫） | 軟骨肉腫（Grade 1） |
|--------|-------------------|-------------------|
| 好発部位 | 手指・手・足の短管骨 | 骨盤・近位長管骨 |
| Endosteal scalloping | 軽度（< 2/3皮質厚） | 深い（> 2/3皮質厚） |
| 骨皮質変化 | 菲薄化のみ（穿孔なし） | 穿孔・破壊あり |
| 軟部腫瘤 | なし | ありえる |
| 疼痛 | なし（偶発的発見が多い） | 疼痛あり（重要な臨床情報） |
| サイズ | 一般に小さい（< 3 cm） | 大きい傾向（> 5 cm） |
| MRI T2 | 均一高信号、点状低信号（石灰化） | 高信号+septal enhancement |

**臨床的疼痛の有無**は軟骨腫と軟骨肉腫の鑑別において最も重要な情報の一つであり、画像所見と合わせて総合的に判断する。

---

## 鑑別診断

| 疾患 | 鑑別のポイント |
|------|---------------|
| **内軟骨腫（Enchondroma）** | 上記参照。短管骨・手足に多い。無症状 |
| **骨肉腫（軟骨芽細胞型）** | 若年者。骨様基質+軟骨基質の混在。骨端板付近に多い |
| **軟骨芽細胞腫** | 骨端部発生。若年者（骨成熟前）。二次性動脈瘤様骨嚢腫を伴うことあり |
| **脱分化型軟骨肉腫** | 低悪性軟骨腫瘍成分+高悪性紡錘形細胞肉腫が共存。画像上「二相性」パターン |
