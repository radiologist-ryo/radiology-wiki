## 概要

骨巨細胞腫（giant cell tumor of bone; GCT）は破骨細胞様多核巨細胞と単核間質細胞からなる腫瘍であり、局所侵攻性を持つが一般に低悪性度と分類される。骨端部（epiphysis）に発生することが最大の特徴である。

- **好発年齢**：20〜40歳代（骨端板閉鎖後）
- **好発部位**：大腿骨遠位・脛骨近位（膝周囲が約50〜60%）・橈骨遠位・仙骨
- **悪性度**：局所侵攻性（再発率20〜40%）。悪性GCT（肉腫への転化）は約1〜2%。肺転移も稀にあり（benign pulmonary implants）
- **分子標的治療**：RANK-Lシグナルを介した破骨細胞活性化が病態の中心であり、デノスマブ（抗RANK-L抗体）による治療が導入されている

## 画像所見

### 単純X線写真

GCTの典型的X線所見は「骨端部の偏在性溶骨性病変」である。<Cite id={1} /><Cite id={2} />

| 所見 | 特徴 |
|------|------|
| 発生部位 | **骨端部**（関節面に接する）。骨端板閉鎖後に発生 |
| 辺縁 | 境界比較的明瞭（Lodwick IB型が多い）。硬化縁は通常なし |
| 基質石灰化 | 通常なし（pure lytic lesion） |
| 骨皮質変化 | 菲薄化・膨隆（balloon-like expansion）。穿孔〜軟部腫瘤（進行例） |
| 骨膜反応 | 通常なし〜軽微 |
| 関節浸潤 | 軟骨下骨を侵食して関節腔に至ることがある |

### Campanacci分類

Campanacci分類は放射線学的侵攻性に基づく予後・治療方針の指標として広く用いられている。<Cite id={1} />

| Grade | 特徴 |
|-------|------|
| **Grade 1** | 境界明瞭・骨皮質正常〜菲薄化のみ |
| **Grade 2** | 境界明瞭・骨皮質菲薄化・膨化（balloon expansion）、穿孔なし |
| **Grade 3** | 骨皮質穿孔・軟部腫瘤形成（最も侵攻性が高い） |

### CT

CTは骨皮質の穿孔・菲薄化の評価と軟部腫瘤の範囲評価に有用である。腫瘍内に石灰化を認めることは稀であり、存在する場合は続発性動脈瘤様骨嚢腫（aneurysmal bone cyst; ABC）の合併を疑う。GCTは約10〜20%にABCの合併を認める。

### MRI

| シーケンス | 所見 |
|-----------|------|
| T1強調像 | 低〜中信号。均一〜軽度不均一 |
| T2強調像 | **低〜中信号**（多核巨細胞内の含鉄血黄素や線維化によりT2信号が低い；他の骨腫瘍と異なる特徴的所見） |
| 造影T1 | 不均一な増強。活性部・壊死部・出血部が混在 |

T2強調像での低信号は多核巨細胞内の含鉄血黄素沈着・線維成分に起因し、GCTの鑑別に重要な特徴である。二次性ABCを合併する場合は液液レベル（fluid-fluid level）を伴うT2高信号嚢胞成分が追加される。

### PET-CT

GCTはヘキソキナーゼ-2の過剰発現によりFDG集積が高く、悪性腫瘍との鑑別が困難な偽陽性を呈することがある。<Cite id={3} /> FDG-PETはGCTに対して特異度が低く、解釈に注意が必要である。

## 鑑別診断

| 疾患 | 鑑別のポイント |
|------|---------------|
| **動脈瘤様骨嚢腫（ABC）** | 液液レベル（fluid-fluid level）。GCTより若年（骨端板閉鎖前も含む）。T2高信号が主体 |
| **軟骨芽細胞腫** | 骨端部。若年者（骨成熟前）。軽度の硬化縁あり。T2低信号傾向 |
| **明細胞型軟骨肉腫** | 骨端部発生。軟骨基質石灰化（小さな環状）。稀少 |
| **色素性絨毛結節性滑膜炎（PVNS）** | 関節周囲（関節内原発）。T2低信号（含鉄血黄素）。骨侵食は外側から |
| **転移性骨腫瘍** | 中高年。多発。既知の原発巣。骨端部への孤立性転移との鑑別（腎癌・甲状腺癌は骨端部転移あり） |
