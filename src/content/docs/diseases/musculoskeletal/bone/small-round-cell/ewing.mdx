## 概要

Ewing肉腫（Ewing sarcoma）は神経外胚葉由来の悪性腫瘍であり、EWSR1遺伝子の転座（主にEWSR1-FLI1融合遺伝子：t(11;22)）を特徴とする。原発性悪性骨腫瘍の約16%を占め、骨肉腫に次ぐ頻度である。

- **好発年齢**：5〜25歳（若年者。20歳未満が約80%）
- **好発部位**：長管骨骨幹部（大腿骨・脛骨・腓骨）および骨盤・肋骨。脊椎発生も約10%に見られる
- **臨床症状**：局所の疼痛・腫脹・発熱・炎症反応上昇（骨髄炎との鑑別が問題となる）

## 画像所見

### 単純X線写真

Ewing肉腫の典型的X線所見は「広範な骨破壊・骨形成なし・巨大軟部腫瘤・骨膜反応」の組み合わせである。

| 所見 | 特徴 |
|------|------|
| 辺縁 | 不明瞭（Moth-eaten〜Permeative型；Lodwick II〜III）<Cite id={2} /> |
| 基質石灰化 | 原則なし（骨形成成分を含まない） |
| 骨破壊 | 骨幹部の広範な浸潤性骨破壊。骨皮質の広範な侵食 |
| 骨膜反応 | **Onion skin（積層状）骨膜反応**が特徴的<Cite id={1} />。Codman三角も出現しうる |
| 軟部腫瘤 | 骨破壊の程度に比して著しく大きい軟部腫瘤（Ewing肉腫の特徴） |

Onion skin骨膜反応は急速増殖性腫瘍の骨膜への侵入と反応性骨形成が交互に繰り返されることで形成される。Ewing肉腫で最も特徴的だが、骨肉腫・骨髄炎・ランゲルハンス組織球症でも出現しうる。

### CT

CTでは骨皮質内の虫食い状変化（endosteal erosion）と、骨外に広がる軟部腫瘤を明瞭に評価できる。石灰化はほとんど認められない。壊死による不均一な低吸収域を軟部腫瘤内に認めることがある。

### MRI

Ewing肉腫のMRI評価は術前計画において最も重要であり、腫瘍の骨髄内進展範囲・軟部浸潤・神経血管束への関与を精密に評価する。

| シーケンス | 所見 |
|-----------|------|
| T1強調像 | 骨髄を置換する低信号。広範な骨幹部に及ぶことが多い |
| T2強調像/STIR | 腫瘍は不均一な中〜高信号。軟部腫瘤内の壊死・出血により信号不均一 |
| 造影T1 | 不均一な増強。壊死部は非増強 |
| DWI | 活性腫瘍部で拡散制限。治療効果判定に有用 |

**全身MRI（whole-body MRI）またはPET-CT**は診断時の骨転移・骨外転移の評価に推奨される。

## 鑑別診断

| 疾患 | 鑑別のポイント |
|------|---------------|
| **骨髄炎（急性化膿性）** | 最重要鑑別。発熱・白血球増多。MRIで骨端板をまたぐ浸潤・周囲軟部炎症・液体貯留。臨床・血液検査・生検が確定に必要 |
| **骨肉腫** | 骨端部〜骨幹端部。骨様基質（osteoid matrix）を認める。Sunburst pattern。年齢（10〜20歳代） |
| **ランゲルハンス細胞組織球症（LCH）** | 病変が単発・境界明瞭なことも多い。脊椎椎体圧壊（vertebra plana）。若年者 |
| **リンパ腫** | 中高年。骨幹部。骨破壊が軽微な割に軟部腫瘤が大きい（Ewing肉腫に類似） |
| **転移性神経芽細胞腫** | 5歳未満の幼児。多発骨転移。尿中カテコラミン高値 |
