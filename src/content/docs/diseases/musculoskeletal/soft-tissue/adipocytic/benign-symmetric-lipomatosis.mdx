## 概要

良性対称性脂肪腫症（benign symmetric lipomatosis; BSL）は、**Madelung病（Madelung disease）**または**多発対称性脂肪腫症（multiple symmetric lipomatosis）**とも呼ばれ、頸部や上半身に左右対称性に成熟脂肪組織が増生する稀な疾患である<Cite refs={[2]} />。

- **好発年齢**：30〜60歳台（中高年）
- **性差**：**男性に圧倒的に多い**（男女比15〜30：1）
- **好発部位**：頸部、肩、上背部に左右対称性に脂肪沈着
- **背景疾患**：アルコール多飲歴、ミトコンドリア病との関連が示唆される
- **WHO分類**：adipocytic tumours の良性カテゴリーに含まれる<Cite refs={[1]} />

## 分類（Enzi分類）

| 病型 | 脂肪沈着部位 |
|------|------|
| Type I（馬の首型） | 頸部・肩・上背部 |
| Type II（仮性アスリート型） | 上腕・胸部・腹部 |
| Type III（lipomatous gynecomastia型） | 骨盤・大腿部 |

## 臨床的問題点

- 整容上の問題
- 頸部脂肪沈着が深部に進展した場合、**気道狭窄**を引き起こすことがある
- 末梢神経の圧迫症状（自律神経障害、感覚障害）

## 画像所見

### CT・MRI

- **被膜を有さない**均一な脂肪塊
- 皮下脂肪組織や深部筋層へ**浸潤性に増殖**
- 左右対称性
- 通常の皮下脂肪と同等のCT値（−50〜−150 HU）、MRI信号（T1/T2高信号、脂肪抑制で抑制）
- 脂肪腫と異なり明瞭な被膜・境界なし

### 評価のポイント

- 気道狭窄の有無（特に Type I で深部進展例）
- 大血管・神経の圧迫
- 範囲の広がり（術前計画）

## 鑑別診断

- **多発脂肪腫**: 被膜あり、左右非対称、各々が分離した腫瘤
- **Cushing症候群の脂肪沈着**: 中心性肥満、buffalo hump、内分泌所見
- **HIV関連脂肪異栄養症**: HIV感染歴、抗レトロウイルス療法歴

## 治療

整容目的または機能障害（気道狭窄など）に対して外科的切除または脂肪吸引。アルコール断酒で進行抑制が期待される。再発しやすく、長期経過観察が必要。
