## 概要

脱分化型脂肪肉腫（dedifferentiated liposarcoma; DDLPS）は、高分化型脂肪肉腫（WDLPS）が進行し、脂肪分化を欠く高悪性度の肉腫成分を含むようになった病変である。高分化型脂肪肉腫と同様に**MDM2遺伝子**および**CDK4遺伝子の増幅**を高頻度に認め、WDLPSの自然経過の延長上に位置する連続的な病態と考えられている。<Cite id={1} /><Cite id={2} />

- **好発年齢**：50〜70歳代
- **好発部位**：後腹膜（最多）、次いで四肢深部軟部組織、精索、縦隔
- **悪性度**：高悪性度。遠隔転移をきたしうる

---

## 臨床像

後腹膜発生例では長期間無症状で経過し、巨大な無痛性腫瘤として発見されることが多い。WDLPSの内部または再発時に出現し、再発を繰り返すうちに進行する。**後腹膜発生例では局所再発率が高く**、遠隔転移も生じやすい。脱分化巣に高悪性度成分が多いほど進行は速く、予後は不良である。<Cite id={2} />

## 画像所見

### CT

CTは脱分化型脂肪肉腫の診断において中核となるモダリティである。後腹膜腫瘤の全容評価、脂肪成分と非脂肪成分の分布確認、遠隔転移評価に優れる。

| 所見 | 特徴 |
|------|------|
| 構成 | 脂肪成分（WDLPS領域）＋非脂肪性充実成分（脱分化巣）の混在 |
| 脂肪成分 | 低吸収（-50〜-120 HU）、内部に線維性隔壁 |
| 脱分化成分 | 軟部組織濃度、不均一な造影効果 |
| 二次変化 | 出血・壊死、骨化・石灰化（比較的多い） |
| 周囲臓器 | 圧排・浸潤（腎・膵・腸管・大血管） |

脱分化巣によりWDLPS領域の脂肪成分が圧排されて不明瞭となることがあり、注意を要する。

### MRI

MRIでは脂肪成分と非脂肪成分の識別、脱分化巣の範囲同定、切除計画に有用である。

| シーケンス | 所見 |
|-----------|------|
| T1強調像 | 脂肪成分：高信号、脱分化成分：筋と等〜低信号 |
| T2強調像 | 脂肪成分：高信号、脱分化成分：不均一な高〜中等度信号 |
| 脂肪抑制像 | 脂肪成分の信号が抑制される（脂肪成分の確認） |
| 造影後T1 | 脱分化成分が不均一に強く造影される |
| DWI/ADC | 脱分化成分で拡散制限（ADC低下） |

### 鑑別に資する画像所見

WDLPSと脂肪腫の鑑別、DDLPS診断において、以下の所見は悪性を示唆する。<Cite id={3} /><Cite id={4} />

- 腫瘍サイズ10 cm以上
- 厚い隔壁（>2 mm）
- 脂肪成分の割合が75%以下
- 造影後の非脂肪信号域が筋と比較して高信号
- 結節状の充実成分

## 鑑別診断

| 疾患 | 鑑別のポイント |
|------|---------------|
| **高分化型脂肪肉腫（WDLPS）** | 非脂肪性充実成分に乏しい。連続したスペクトラム上の病態。MDM2/CDK4増幅を共有 |
| **粘液型脂肪肉腫** | T2著明高信号の粘液性基質。FUS::DDIT3融合。後腹膜原発は稀で、後腹膜病変は四肢からの転移または脱分化巣内粘液変化を疑う |
| **多形型脂肪肉腫** | 純粋な高悪性度肉腫。WDLPS領域を伴わない |
| **平滑筋肉腫** | 後腹膜大血管原発（下大静脈・卵巣静脈）。脂肪成分なし |
| **未分化多形肉腫（UPS）** | 脂肪成分を欠く。MDM2増幅陰性。組織学的鑑別が必要 |
| **後腹膜線維症** | 対称性・大動脈周囲のびまん性肥厚。腫瘤形成なし |

## 治療

治療の中心は**広範切除によるR0切除**である。後腹膜発生例では切除範囲の設定が困難で、近接臓器の合併切除が必要となることが多い。化学療法の感受性は限定的で、ドキソルビシン単剤またはAI療法（ドキソルビシン+イホスファミド）が選択される。再発・難治例には、**エリブリンメシル酸塩**（微小管阻害薬）が腫瘍内部変化を指標に有効性が報告されている。

---

## 画像診断のポイント

1. **後腹膜の大型脂肪性腫瘤を見たら、必ず非脂肪成分の有無を確認する。**
2. 脂肪成分の中に結節状・塊状の非脂肪性充実成分があれば脱分化を示唆する。
3. 造影効果の強い領域・骨化を伴う領域は脱分化巣の可能性が高く、生検ターゲットとなる。
4. 治療効果判定ではサイズ変化だけでなく、**壊死の程度・造影効果の低下・ADC上昇**を含む質的評価が重要（Choi基準）。
