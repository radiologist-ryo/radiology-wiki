## 概要

粘液型脂肪肉腫（myxoid liposarcoma; MLPS）は、豊富な粘液状基質と網目状の分枝血管（chicken-wire vasculature）を特徴とする脂肪肉腫で、**FUS::DDIT3融合遺伝子**（稀に**EWSR1::DDIT3**）を有することが診断の決め手となる。<Cite id={1} />

- **好発年齢**：30〜50歳代（AYA世代も含む）
- **好発部位**：四肢深部軟部組織、特に**大腿**（最多）。体幹・後腹膜原発は稀
- **悪性度**：中等度〜高度（円形細胞成分の比率で予後が決まる）

---

## 臨床像・転移パターン

無痛性腫瘤として発見されることが多い。**局所再発や遠隔転移を起こしやすく、転移部位が特徴的**で、肺のみならず**後腹膜・縦隔・脊椎・他軟部組織**への転移が多い。骨転移では造骨性変化を示さず、骨シンチグラフィで検出しにくいため、全身MRIが有用である。<Cite id={3} />

**後腹膜に粘液型脂肪肉腫様の腫瘤を認めた場合**は、以下を考慮すべきである。

- 四肢原発からの後腹膜転移
- 脱分化型脂肪肉腫の粘液性変化
- 純粋な後腹膜原発MLPSは稀

## 画像所見

### CT

CTでは粘液成分を反映して**筋よりやや低吸収**（水に近い）の分葉状腫瘤として描出される。内部に脂肪成分を含むことがあり、その場合は低吸収域として認められる。造影では不均一な造影効果を示す。

### MRI

MRIが診断の主役となる。粘液性基質の豊富さを反映した特徴的な信号パターンを示す。

| シーケンス | 所見 |
|-----------|------|
| T1強調像 | 筋と同程度の**低〜等信号**。一部に脂肪成分を反映した高信号域が混在することがある |
| T2強調像 | **著明な高信号**（粘液基質を反映）、時に水腫と見紛う |
| 脂肪抑制T2/STIR | 高信号が維持される |
| 造影後脂肪抑制T1 | 不均一で比較的強い造影効果（水腫との鑑別点） |
| DWI/ADC | 円形細胞成分の多い部位で拡散制限を示す |

**分葉状の形態・隔壁構造・豊富な造影効果**が嚢胞性病変との鑑別点となる。純粋なMLPSでは内部に少量の脂肪成分（T1高信号域）を含むことがあり、診断の手がかりとなる。

### 病理学的悪性度と画像の対応

MLPSは円形細胞成分の割合で悪性度が決まる。

- **低悪性度型**：粘液成分優位、T2著明高信号、予後良好
- **高悪性度型**：円形細胞成分 >5%、充実成分主体、予後不良、転移リスク高

画像上、T2強調像で**低信号の充実成分**が目立つ場合は円形細胞成分が多い可能性があり、悪性度が高いと判断される。

## 鑑別診断

| 疾患 | 鑑別のポイント |
|------|---------------|
| **筋肉内粘液腫（intramuscular myxoma）** | 良性。造影効果に乏しい。脂肪成分なし。周囲に脂肪の縁取り（fat cap） |
| **神経鞘腫** | target sign、神経との連続性、split fat sign |
| **高分化型脂肪肉腫** | 脂肪成分主体、厚い隔壁、非脂肪成分は少量 |
| **滑膜肉腫** | 石灰化を伴うことがある。二相性信号パターン（triple sign） |
| **軟部転移（特に後腹膜）** | 原発巣の検索が必要。後腹膜MLPS様腫瘤では四肢原発の検索を忘れない |
| **粘液線維肉腫** | 高齢者、皮下・筋膜下に多い、tail sign |

## 治療

治療の中心は**広範切除**である。放射線治療に対する感受性が比較的高く、術前・術後照射が併用される。化学療法はドキソルビシン単剤やAI療法が用いられる。

**トラベクテジン**（DNA副溝結合薬）は粘液型脂肪肉腫・滑膜肉腫などの転座関連肉腫に対して高い有効性が示されており、腫瘍サイズは不変でも造影効果が低下し、病理組織学的に高度壊死を示す反応様式が報告されている。<Cite id={4} />

## 画像診断のポイント

1. **四肢深部（特に大腿）のT2著明高信号腫瘤** → 粘液型脂肪肉腫を最優先に想起
2. 水腫性病変との鑑別には**造影効果の確認**が必須
3. T2強調像での**低信号充実成分の多さ** = 円形細胞成分が多く、高悪性度を示唆
4. 転移検索では**後腹膜・縦隔・脊椎・他軟部組織**を含む**全身MRI**が望ましい
5. 術前照射後の治療効果判定では、**サイズ変化よりも造影効果減弱・ADC上昇**が重要
