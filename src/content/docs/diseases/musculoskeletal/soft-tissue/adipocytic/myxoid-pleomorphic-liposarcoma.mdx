## 概要

粘液多形型脂肪肉腫（myxoid pleomorphic liposarcoma）は、WHO骨軟部腫瘍分類（第5版）で**新たに悪性脂肪性腫瘍として追加された**稀な高悪性度腫瘍である。<Cite id={1} />

組織所見は粘液型脂肪肉腫に類似するが、**腫瘍細胞の多形性が目立つ**点で異なる。粘液型脂肪肉腫で特異的な**FUS::DDIT3（あるいはEWSR1::DDIT3）融合遺伝子を欠く**ことが分子診断上の決め手となる。<Cite id={1} /><Cite id={2} />

- **好発年齢**：**女児および若年成人女性**
- **性別**：女性優位
- **好発部位**：**縦隔**、**大腿**、**頭頸部**
- **悪性度**：高悪性度、予後不良

---

## 臨床像

若年女性の縦隔発生例が特徴的である。縦隔病変は長期間無症状で経過し、巨大化してから胸部症状（呼吸困難、咳嗽、上大静脈症候群）で発見されることが多い。

**予後は不良**で、遠隔転移を高頻度にきたす。

---

## 分子遺伝学的特徴

- **FUS::DDIT3融合遺伝子を欠く**（粘液型脂肪肉腫との鑑別点）
- **EWSR1::DDIT3融合遺伝子も欠く**
- RB1遺伝子欠失、染色体数異常など複雑な変化を示す

---

## 画像所見

本疾患は極めて稀なため、画像所見の系統的報告は限られている。粘液型脂肪肉腫と類似した所見を示すが、より**多形性・不均一性が強く**、発生部位（特に縦隔）が異なる点が特徴である。

### CT

| 所見 | 特徴 |
|------|------|
| 形態 | 大型の分葉状腫瘤、境界比較的明瞭 |
| 内部 | 粘液成分による低吸収域＋充実成分の混在 |
| 造影効果 | 不均一な造影効果 |
| 二次変化 | 壊死・出血を伴うことあり |

### MRI

| シーケンス | 所見 |
|-----------|------|
| T1強調像 | 筋と等〜低信号、脂肪成分を含む場合は高信号域 |
| T2強調像 | 粘液成分を反映した高信号、充実成分は中等度〜高信号で混在 |
| 脂肪抑制T2 | 高信号が維持される |
| 造影後T1 | **不均一で強い造影効果**（粘液型脂肪肉腫より強い傾向） |
| DWI/ADC | 充実成分で拡散制限 |

## 鑑別診断

| 疾患 | 鑑別のポイント |
|------|---------------|
| **粘液型脂肪肉腫** | **FUS::DDIT3（またはEWSR1::DDIT3）融合陽性**、多形性は軽度。若年〜中年、四肢深部好発 |
| **多形型脂肪肉腫** | 高齢者、粘液成分に乏しい、純粋な高悪性度肉腫 |
| **脱分化型脂肪肉腫** | 後腹膜好発、MDM2/CDK4増幅、WDLPS領域を伴う |
| **未分化多形肉腫** | 脂肪成分を欠く、高齢者 |
| **縦隔の奇形腫・胚細胞腫瘍** | 縦隔発生の若年女性での鑑別。AFP・hCG・血清マーカー |

---

## 治療と予後

- **治療**：広範切除が基本。縦隔発生例では完全切除が困難なことが多い
- **化学療法**：粘液型脂肪肉腫と異なりトラベクテジンへの反応は期待しにくい。ドキソルビシンベースのレジメン
- **放射線治療**：補助療法として併用
- **予後**：高悪性度で**予後不良**、早期遠隔転移のリスクが高い

---

## 画像診断のポイント

1. **若年女性の縦隔の脂肪成分を含む大型腫瘤** → 粘液多形型脂肪肉腫を鑑別に挙げる
2. 粘液型脂肪肉腫と画像所見が類似するため、**発生部位・年齢・性別**と**分子診断**が鑑別に必須
3. WHO第5版以前は粘液型脂肪肉腫の多形成分が強い例として扱われていた可能性がある
4. 診断には**病理学的検討＋分子遺伝学的検索（FUS::DDIT3陰性の確認）**が不可欠
