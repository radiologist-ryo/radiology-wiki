## 概要

高分化型脂肪肉腫（well-differentiated liposarcoma; WDL）は、成熟脂肪細胞の増生を主体とし、脂肪細胞と線維性間質細胞に核異型を伴う悪性軟部腫瘍である。軟部肉腫のうち最も頻度が高い脂肪性腫瘍群に属し、ほぼ全例で **MDM2遺伝子・CDK4遺伝子の増幅** が認められる。<Cite id={1} /><Cite id={2} />

発生部位により呼称が異なり、**四肢など外科的切除で根治が期待できる部位に発生したもの** を **異型脂肪腫様腫瘍（atypical lipomatous tumour; ALT）**、**縦隔・後腹膜など完全切除困難な部位に発生したもの** を WDL と呼ぶ。本質的には同一疾患であり、予後は局在に強く依存する。

- **好発年齢**：50〜70歳代
- **好発部位**：四肢近位部（大腿など）・体幹深部・後腹膜・縦隔・精索
- **遺伝子異常**：MDM2・CDK4増幅（ほぼ全例、FISHで診断可能）
- **転移**：稀（脱分化を伴う場合を除く）
- **予後**：四肢例は局所再発少・低死亡率／**後腹膜例はほぼ全例再発、長期経過で80%以上が腫瘍死** <Cite id={2} />

## 組織学的亜型

| 亜型 | 特徴 |
|------|------|
| **脂肪腫様亜型（lipoma-like）** | 成熟脂肪細胞主体。良性脂肪腫との鑑別が最も困難 |
| **硬化亜型（sclerosing）** | 線維性間質が豊富。非脂肪成分が目立つ |
| **炎症亜型（inflammatory）** | リンパ球・形質細胞浸潤を伴う。後腹膜に多い |

## 画像所見

### 共通所見

CT・MRIともに脂肪と同等の信号・吸収値を主体とし、内部に **様々な厚さの線維性隔壁** を伴う **分葉状腫瘤** として描出される。亜型により非脂肪成分の比率・信号が多彩となる。画像上、脂肪成分のみにみえる場合は良性脂肪腫との鑑別が困難なことが多い。

### CT

| 所見 | 特徴 |
|------|------|
| 主成分 | 脂肪吸収値（−50〜−150 HU） |
| 隔壁 | 様々な厚さの線維性隔壁。造影で軽度〜中等度増強 |
| 非脂肪成分 | 硬化亜型で目立つ（結節状領域） |
| 形態 | 分葉状・被膜不明瞭 |
| 石灰化 | 稀（伴う場合は脱分化を疑う） |

### MRI

| シーケンス | 所見 |
|-----------|------|
| T1強調像 | 脂肪と同等の高信号（主成分）＋隔壁の低〜中信号 |
| 脂肪抑制T2 | 脂肪成分の信号が抑制され、隔壁・非脂肪成分が高信号で残る |
| 造影脂肪抑制T1 | 隔壁および非脂肪結節が軽度〜中等度増強 |

非脂肪成分が 2 cm 以上の結節状で造影増強を示す場合、**脱分化（dedifferentiation）** への移行を強く疑う。

### 後腹膜型の特徴

後腹膜発生例は無症状で経過し、**巨大腫瘤として偶発的に発見されることが多い**（体重増加の自覚で精査→巨大脂肪性腫瘍判明、のパターン）。完全切除が困難で、ほぼ全例で局所再発を繰り返し、再発ごとに脱分化成分を伴うリスクが上昇する。

## 鑑別診断

| 疾患 | 鑑別のポイント |
|------|---------------|
| **脂肪腫（lipoma）** | 厚い隔壁なし・非脂肪結節なし・2 cm超の非脂肪領域なし。MDM2 FISH陰性 |
| **脱分化型脂肪肉腫** | 脂肪成分＋高悪性度非脂肪成分（不均一造影・壊死）。連続病態として発生 |
| **粘液型脂肪肉腫** | 四肢深部。T2著明高信号（粘液基質）。脂肪成分は部分的 |
| **血管脂肪腫** | 小型皮下。疼痛を伴う。血管成分 |
| **脂肪芽腫（小児）** | 乳幼児。自然退縮傾向 |

## 診断上のポイント

- **MDM2 FISH** は ALT/WDL と良性脂肪腫の鑑別に決定的
- 後腹膜の脂肪性腫瘍は **ほぼ全例が WDL または脱分化型** と考える <Cite id={2} />
- 非脂肪結成が 2 cm 以上・造影効果強い → 脱分化への移行を疑う
