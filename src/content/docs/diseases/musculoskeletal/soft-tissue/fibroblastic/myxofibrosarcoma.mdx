## 概要

粘液線維肉腫（myxofibrosarcoma; MFS）は、粘液性間質を背景に多様な細胞密度と形態異型を示す悪性線維芽細胞性腫瘍である。かつては"粘液型悪性線維性組織球腫（myxoid malignant fibrous histiocytoma; MFH）"と呼ばれていた。<Cite id={1} />

- **好発年齢**：50〜70歳代（高齢者に多い）
- **性別**：男性にやや多い
- **好発部位**：四肢（特に下肢）の**皮下〜筋膜下**（約50%）、残りは筋膜下や筋内
- **特徴**：**浸潤型腫瘍の代表疾患**。肉眼的辺縁を越えて浸潤性に広がる

後腹膜発生は稀で、その場合は脱分化型脂肪肉腫との鑑別が問題となる。

---

## 臨床像

緩徐に増大する無痛性腫瘤として発見されることが多い。**しばしば多結節性で浸潤性に広がり**、臨床的に触知する範囲より広く進展していることが多い。切除後の**局所再発率が30〜40%**と高く、その主因は断端陽性である。<Cite id={2} /> 遠隔転移は主に肺・骨に生じる。

## 画像所見

### MRI（診断・切除計画の主役）

MRIで**tail-like sign**（腫瘍辺縁から筋膜に沿って伸びる浸潤性の発育）の同定が最も重要である。<Cite id={3} />

| シーケンス | 所見 |
|-----------|------|
| T1強調像 | 腫瘤：筋と等〜低信号。浸潤部分は同定困難 |
| T2強調像 | **著明な高信号**（粘液性間質を反映）。高信号域は反応層＋浸潤範囲を含む |
| 脂肪抑制T2/STIR | 浸潤部分が高信号として描出される |
| 造影後脂肪抑制T1 | **浸潤部分が筋膜に沿った線状造影効果（tail sign）として描出**される |
| DWI/ADC | 細胞密度の高い部位で拡散制限 |

### tail-like signの重要性

- **造影後脂肪抑制T1強調像でのtail sign** = 病理学的浸潤範囲をよく反映する
- **T2強調像での高信号域** = 反応層（腫瘍細胞を含まない浮腫）に相当
- 浸潤範囲は造影で描出される範囲 < T2高信号域
- 切除縁計画では**造影所見を基準に、通常よりも広い切除範囲**を設定する必要がある

### CT

CTは胸部転移検索や深在性病変の評価に用いられる。粘液成分により筋と等〜軽度低吸収を呈し、不均一に造影される。

## 浸潤型腫瘍の切除縁設定

粘液線維肉腫と未分化多形肉腫（UPS）は浸潤型腫瘍の代表であり、通常の広範切除ではR0切除が達成できないことがある。<Cite id={3} /><Cite id={2} />

**切除計画のポイント**

1. **造影後脂肪抑制T1強調像でのtail signを忠実にマッピング**し、皮膚切開・切除範囲に反映
2. T2高信号域はtail signより広い範囲に及ぶため、浸潤範囲と反応層の区別が重要
3. 前腕・下腿など機能を考慮すると切除範囲拡大が困難な部位では、**化学療法・放射線治療との集学的治療**で局所制御と患肢温存を両立

術前・術後放射線治療は、浸潤型腫瘍のR0切除が困難な場合に併用される。

## 鑑別診断

| 疾患 | 鑑別のポイント |
|------|---------------|
| **粘液型脂肪肉腫** | 若年〜中年、四肢深部（大腿）、脂肪成分を含むことあり、FUS::DDIT3融合 |
| **筋肉内粘液腫** | 良性、造影効果乏しい、周囲浮腫なし、fat cap |
| **未分化多形肉腫（UPS）** | 粘液成分に乏しい、より充実性、高齢者に多い。浸潤型という点では共通 |
| **結節性筋膜炎** | 急速増大後に自然退縮、若年〜中年、筋膜に接する小型腫瘤 |
| **皮下悪性軟部腫瘍（脱分化型脂肪肉腫の皮下進展など）** | 既往歴・深部病変の確認 |

---

## 治療

1. **広範切除によるR0切除**が根治の基本。浸潤範囲を考慮した十分なマージン設定
2. 高悪性度例・切除困難例には**術前・術後化学療法**（ドキソルビシン、AI療法）
3. **術前・術後放射線治療**の併用
4. 再発・難治例には、パゾパニブ、エリブリン、トラベクテジンなどの分子標的薬・新規抗癌薬

---

## 画像診断のポイント

1. **高齢者の四肢（特に下肢）皮下〜筋膜下のT2高信号腫瘤** → 粘液線維肉腫を最優先
2. **造影後脂肪抑制T1強調像でのtail sign** は診断・切除計画の最重要所見
3. T2高信号域の全範囲を切除対象とするのは過大切除。造影所見で浸潤範囲を判断
4. 多結節性発育を見たら浸潤型腫瘍を疑う
5. **再発例の再評価**ではさらに広範な浸潤を示すことがあり、注意深い再MRIが必須
