## 概要

孤立性線維腫（solitary fibrous tumour; SFT）は、**NAB2::STAT6融合遺伝子**を分子学的背景とする線維芽細胞性腫瘍で、全身のあらゆる部位に発生しうる。従来"血管外皮腫（hemangiopericytoma）"と呼ばれていた腫瘍の多くが、現在ではSFTに包含されている。<Cite id={1} />

- **好発年齢**：40〜70歳代
- **好発部位**：胸膜（最多）、腹腔・後腹膜、四肢軟部、頭頸部、眼窩、髄膜、骨
- **悪性度**：大部分は良性〜中間性だが、一部に悪性（metastasizing）例あり

免疫染色で**STAT6核内陽性**が診断的であり、CD34もしばしば陽性となる。<Cite id={1} />

---

## 臨床像

緩徐に増大する**無痛性腫瘤**として発見されることが多い。腹腔内・後腹膜発生例では長期間無症状で経過し、偶発的に発見される。胸膜発生例では気胸・胸水・ばち指（Pierre-Marie-Bamberger症候群）を伴うことがある。

一部の大型例では**Doege-Potter症候群**（腫瘍産生IGF-2による低血糖）をきたすことがあり、臨床上の注意点となる。

## 画像所見

### CT

CTでは境界明瞭な充実性腫瘤として描出され、**強い造影効果**が特徴的である。

| 所見 | 特徴 |
|------|------|
| 形態 | 境界明瞭、分葉状、多くは充実性 |
| 造影前 | 筋と等〜軽度高吸収 |
| 造影効果 | **動脈相で強く不均一に造影**される（hypervascular）|
| 二次変化 | 大型病変では内部に出血・嚢胞変性・壊死 |
| 石灰化 | 稀だが大型例で認めうる |
| 周囲臓器 | 圧排主体、浸潤は稀 |

### MRI

MRIでは線維性間質と豊富な血管を反映した信号パターンを示す。

| シーケンス | 所見 |
|-----------|------|
| T1強調像 | 筋と等〜軽度低信号 |
| T2強調像 | **混在信号**：線維成分の低信号と細胞成分の高信号が混在（heterogeneous） |
| 脂肪抑制T2 | 線維成分：低信号、細胞成分：高信号 |
| 造影後T1 | 強く不均一に造影、flow voidを伴うことあり |
| DWI | 細胞密度の高い領域で拡散制限 |

大型病変（特に胸膜・腹腔発生例）では嚢胞変性・出血を高頻度に伴い、不均一な内部構造として描出される。

## 悪性度の予測

SFTの悪性度予測には**Demicco分類**（年齢・腫瘍サイズ・核分裂像・壊死を組み合わせたリスクスコア）が用いられる。画像上では以下の所見が悪性を示唆する。

- 腫瘍サイズ 10 cm以上
- 内部の壊死・出血が顕著
- 境界不明瞭・周囲浸潤
- 短期間での急速な増大
- 遠隔転移（肺・骨・肝が多い）

## 鑑別診断

### 胸膜発生

| 疾患 | 鑑別のポイント |
|------|---------------|
| **悪性胸膜中皮腫** | びまん性胸膜肥厚、境界不明瞭、胸水を伴う |
| **胸膜転移** | 多発、原発癌の既往 |
| **神経原性腫瘍** | 椎体傍発生、肋間神経沿い、拡張肋間孔 |

### 腹腔・後腹膜発生

| 疾患 | 鑑別のポイント |
|------|---------------|
| **GIST** | 消化管由来、KIT陽性 |
| **平滑筋肉腫** | 大血管原発、壊死顕著 |
| **脂肪肉腫** | 脂肪成分あり |
| **神経内分泌腫瘍** | ソマトスタチン受容体シンチで集積 |

### 四肢軟部発生

| 疾患 | 鑑別のポイント |
|------|---------------|
| **筋肉内粘液腫** | T2著明高信号、造影効果乏しい |
| **神経鞘腫** | target sign、神経との連続性 |
| **血管平滑筋腫** | 皮下〜筋内、境界明瞭 |

## 治療

治療の中心は**広範切除**である。**NAB2::STAT6融合陽性**であればSTAT6免疫染色で診断確定できる。

悪性例や切除不能例には、**パゾパニブ塩酸塩**（チロシンキナーゼ阻害薬）が無増悪生存期間の延長に有効との報告がある。<Cite id={2} /> 他にダカルバジン、エリブリン、トラベクテジンなども選択肢となる。

---

## 画像診断のポイント

1. **あらゆる部位で発生しうる**ため、胸膜・腹腔・後腹膜・四肢軟部の充実性腫瘤の鑑別に常に挙げる
2. **強い不均一な造影効果** = hypervascular tumor。血管造影でも豊富な腫瘍血管を描出
3. T2強調像での**線維成分の低信号と細胞成分の高信号の混在**が示唆的
4. STAT6免疫染色陽性・NAB2::STAT6融合確認が診断確定に必須
5. 長期経過での**晩期再発・転移**が知られており、長期フォローアップが推奨される
