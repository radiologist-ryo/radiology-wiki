## 概要

悪性軟部腫瘍（軟部肉腫；soft tissue sarcoma）は、発生部位・組織型ともにきわめて多様で、わが国では**希少がん**（罹患率10万人当たり6例未満）に該当する。粗罹患率は5.3／10万人とされる。

軟部肉腫の画像診断では、確定診断が困難な症例も多く、**「良性と判断してよいか」「次のステップ（生検・追加検査）に進むべきか」**の見極めが求められる。最終診断は病理組織学的検査が不可欠で、画像と病理が矛盾しないかの確認が重要である。

## わが国の軟部肉腫の頻度

『軟部腫瘍診療ガイドライン2020』によると、頻度の高い順に：

1. 脂肪肉腫（最多）
2. 粘液線維肉腫
3. 平滑筋肉腫
4. 滑膜肉腫

## WHO第5版における大分類

『WHO骨軟部腫瘍分類（第5版）』では、発生起源・分化方向に基づいて以下のように分類される<Cite refs={[1]} />:

| 大分類 | 主な悪性腫瘍 |
|--------|--------------|
| 脂肪性腫瘍 | 高分化型・脱分化型・粘液型・多形型・粘液多形型脂肪肉腫 |
| 線維芽細胞性/筋線維芽細胞性腫瘍 | 線維肉腫、粘液線維肉腫、UPS、孤立性線維腫（悪性） |
| 血管性腫瘍 | 血管肉腫、類上皮血管内皮腫 |
| 平滑筋性腫瘍 | 平滑筋肉腫 |
| 骨格筋性腫瘍 | 横紋筋肉腫（胎児型・胞巣型・多形性・紡錘形/硬化性） |
| 軟骨・骨形成性腫瘍 | 骨外性骨肉腫 |
| 末梢神経腫瘍 | 悪性末梢神経鞘腫瘍（MPNST）、悪性黒色神経鞘腫瘍 |
| 発生起源・分化方向不明腫瘍 | 滑膜肉腫、類上皮肉腫、胞巣状軟部肉腫、明細胞肉腫、骨外性粘液型軟骨肉腫 など |
| 骨軟部未分化小円形細胞肉腫（独立カテゴリー） | Ewing肉腫、CIC再構成肉腫、BCOR肉腫、EWSR1-non-ETS肉腫 |

各分類は良性・中間悪性（局所侵襲性、低頻度転移性）・悪性に細分化される。

## 画像診断アプローチ

### CT

- ステージング（肺転移、リンパ節転移、骨転移）
- 後腹膜・骨盤発生例での全体像評価
- 内部の石灰化・脂肪・出血の検出

### MRI（最も重要）

#### 評価項目

1. **腫瘍の局在と進展範囲**: 神経血管浸潤、骨浸潤、関節内浸潤
2. **内部性状**: T1/T2信号パターン、脂肪成分、嚢胞・壊死、出血、石灰化
3. **造影パターン**: 均一/不均一、強い造影効果、辺縁優位 vs 内部
4. **拡散制限**: 細胞密度の評価
5. **反応層**: 腫瘍周囲の浮腫、tail-like sign

#### 主要シーケンスと所見

- **T1強調像**: 脂肪成分の検出に最適、出血の評価
- **T2強調像**: 粘液成分は著明な高信号、石灰化は低信号
- **脂肪抑制T2強調像**: 反応層の評価
- **造影後脂肪抑制T1強調像**: 浸潤範囲の評価、tail-like sign

### Tail-like sign（浸潤型腫瘍）

**粘液線維肉腫**や **未分化多形肉腫（UPS）** などの浸潤型腫瘍では、肉眼的辺縁から広く浸潤する性質がある。

- 造影後脂肪抑制T1強調像で、腫瘍辺縁から筋膜に沿って伸びる線状造影効果（**tail-like sign**）
- 病理学的浸潤範囲を反映
- 通常の広範切除でも R0切除困難で、**通常より広い切除範囲**が必要<Cite refs={[2]} />

## 治療効果判定

### RECIST基準
従来の標準。サイズ評価のみ。

### Choi基準（軟部肉腫で重要）
化学療法（特に分子標的薬）による腫瘍縮小効果が乏しい場合でも、内部性状変化（**壊死・造影効果の低下**）から治療効果を評価<Cite refs={[3]} />:

- **PR（partial response）**: サイズ10%以上減少 OR CT値15%以上減少
- 内部の壊死・造影効果消失が病理的壊死を反映

### iRECIST
免疫療法評価。pseudo-progression（一時的増大後の縮小）に対応。

### 質的画像評価の重要性

近年、分子標的薬（パゾパニブ、トラベクテジン、エリブリン）や免疫チェックポイント阻害薬の登場で、サイズ不変でも内部変化（壊死、造影低下、ADC上昇）が治療効果を反映する症例が増加。**形態評価のみでなく質的画像評価**が重要となっている。

## 多職種連携（MDT）における放射線科医の役割

| 診療フェーズ | 画像診断医の主な役割 |
|--------------|----------------------|
| 初期評価 | 局在・解剖学的広がり、神経血管浸潤、転移評価、術前シミュレーション |
| 治療経過中 | 化学療法効果判定（サイズ、壊死、造影効果、ADC値）、pseudo-progressionの鑑別、放射線治療計画用の腫瘍境界明示 |
| 治療後経過観察 | 再発・転移の早期検出、治療後変化（線維化、壊死）と再発の鑑別、晩期障害の評価 |

## 鑑別診断の基本

### 良性 vs 悪性の鑑別ポイント

- サイズ（5cm以上は悪性疑い増加）
- 増大速度（急速増大は悪性疑い）
- 深部 vs 表在（深部発生は悪性疑い）
- 内部の不均一性（出血、壊死は悪性傾向）
- 拡散制限の強さ
- 周囲浸潤所見

### "良性と判断してよい" 所見

- 著明な T2 高信号で均一（嚢胞、粘液性）
- 脂肪信号で均一（脂肪腫）
- 神経との連続性 + target sign（神経鞘腫）
- 関節との連続性（ガングリオン）
- 筋膜に沿う扁平な腫瘤（結節性筋膜炎）

ただし最終的には**画像所見だけでは確定診断困難**であることが多く、**生検**が必要となる症例も多い。
