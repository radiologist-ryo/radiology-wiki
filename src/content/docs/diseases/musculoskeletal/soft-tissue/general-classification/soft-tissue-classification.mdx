## 概要

軟部腫瘍は骨格筋・脂肪・線維組織・血管・末梢神経などの間葉系組織に発生する腫瘍群で、細胞の分化方向に基づいて分類される。ただし対応する正常組織が存在しない腫瘍も少なくなく、滑膜肉腫のように分化が明確でないものは"分化不明腫瘍（tumours of uncertain differentiation）"として分類される。<Cite id={1} />

『WHO骨軟部腫瘍分類（第5版）』（2020年刊行）では、旧来の形態学的腫瘍名に加え、変異遺伝子名や特異的な免疫染色名を冠する新規腫瘍が増加した。また軟部腫瘍と骨腫瘍にそれぞれ記載されていた小円形細胞肉腫は、新設された"骨軟部未分化小円形細胞肉腫"という独立カテゴリーに統合された。<Cite id={1} /><Cite id={2} />

## WHO第5版における軟部腫瘍の大分類

分類は"良性（benign）"、"中間性（intermediate）"、"悪性（malignant）"に大別される。中間性はさらに**局所侵襲性（locally aggressive）**と**低頻度転移性（rarely metastasizing）**の2つに細分される。<Cite id={1} />

| 大分類 | 代表疾患 |
|--------|---------|
| 脂肪性腫瘍（adipocytic） | 脂肪腫、高分化型脂肪肉腫、脱分化型脂肪肉腫、粘液型脂肪肉腫、多形型脂肪肉腫、異型紡錘形細胞/多形脂肪腫様腫瘍（新） |
| 線維芽細胞性/筋線維芽細胞性（fibroblastic/myofibroblastic） | 結節性筋膜炎、デスモイド型線維腫症、孤立性線維腫、粘液線維肉腫、未分化多形肉腫 |
| いわゆる線維組織球性（fibrohistiocytic） | 腱鞘滑膜巨細胞腫、深在性線維組織球腫 |
| 血管性（vascular） | 血管腫、カポジ様血管内皮腫、血管肉腫 |
| 血管周皮腫性（pericytic） | 血管平滑筋腫、グロームス腫瘍 |
| 平滑筋性（smooth muscle） | 平滑筋腫、平滑筋肉腫、EBV関連平滑筋腫瘍（新）、炎症性平滑筋肉腫（新） |
| 骨格筋性（skeletal muscle） | 横紋筋腫、胎児型/胞巣型/多形性/紡錘形細胞型横紋筋肉腫 |
| 末梢神経（peripheral nerve sheath） | 神経鞘腫、神経線維腫、悪性末梢神経鞘腫瘍（MPNST） |
| 分化不明（uncertain differentiation） | 滑膜肉腫、類上皮肉腫、胞巣状軟部肉腫、明細胞肉腫、骨外性粘液型軟骨肉腫、NTRK再構成紡錘形細胞腫瘍（新） |
| 骨軟部未分化小円形細胞肉腫（新設） | Ewing肉腫、CIC再構成肉腫、BCOR遺伝子異常肉腫、EWSR1非ETS融合型円形細胞肉腫 |

## WHO第5版で新規追加された主な軟部腫瘍

第5版では8種類以上の腫瘍が新たな疾患概念として収載された。そのうち、**遺伝子名を冠するもの2種類**、**免疫染色抗体名を冠するもの1種類**、**ウイルス名を冠するもの1種類**があり、従来の細胞分化による分類の枠を越えた命名法が特筆される。<Cite id={1} /><Cite id={2} />

- **異型紡錘形細胞/多形脂肪腫様腫瘍（atypical spindle cell/pleomorphic lipomatous tumour; ASPLT）** — 良性脂肪性腫瘍
- **粘液多形型脂肪肉腫（myxoid pleomorphic liposarcoma）** — 若年女性に好発、予後不良
- **EWSR1-SMAD3陽性線維芽細胞性腫瘍** — 良性
- **軟部血管線維腫（angiofibroma of soft tissue）** — AHRR::NCOR2融合、良性
- **表在性CD34陽性線維芽細胞性腫瘍** — 中間性（低頻度転移性）
- **EBV関連平滑筋腫瘍** — 免疫不全患者に発症
- **炎症性平滑筋肉腫** — 通常型より予後良好
- **NTRK再構成紡錘形細胞腫瘍** — TRK阻害薬が有効

また、**悪性線維性組織球腫（MFH）**の名称は第5版で完全に消滅し、現在は未分化多形肉腫（UPS）や粘液線維肉腫に再分類されている。

## 疫学

悪性軟部腫瘍（軟部肉腫）の粗罹患率はわが国で人口10万人当たり約5.3人とされ、全悪性腫瘍の約1%未満を占める"希少がん"である。<Cite id={4} />

『軟部腫瘍診療ガイドライン2020』および全国軟部腫瘍登録によると、頻度の高い軟部肉腫は以下の順である。<Cite id={4} />

1. 未分化多形肉腫（UPS）
2. 高分化型脂肪肉腫
3. 粘液型脂肪肉腫
4. 平滑筋肉腫
5. 粘液線維肉腫
6. 滑膜肉腫
7. 悪性末梢神経鞘腫瘍
8. 脱分化型脂肪肉腫

脂肪性腫瘍は軟部肉腫全体の中で最大グループを形成し、画像診断においてはその鑑別が最も重要となる。

## 画像診断の原則

軟部腫瘍の画像診断では、**年齢・性別・発生部位・腫瘍サイズ・増大速度・硬さ・可動性**といった臨床情報と、MRI所見を総合的に評価する。画像のみでの確定診断が困難なことも多く、最終的な確定診断には病理組織学的検査が不可欠である。<Cite id={3} />

### MRI撮像の原則

| シーケンス | 役割 |
|-----------|------|
| T1強調像 | 脂肪成分・出血の同定、腫瘍と正常組織の境界 |
| T2強調像 | 粘液性基質・壊死・嚢胞変性の評価 |
| 脂肪抑制T2強調像/STIR | 浮腫・反応層の評価 |
| 造影後脂肪抑制T1強調像 | 充実成分・壊死部の区別、浸潤範囲 |
| 拡散強調像（DWI）/ADC | 悪性度評価の補助（細胞密度反映） |

### 画像診断医の判断ポイント

特に臨床医から求められるのは以下である。<Cite id={3} />

- 「良性と判断してよいか」
- 「次の診断ステップ（生検・切除）に進むべきか」
- 「切除縁計画に資する浸潤範囲の明示」

画像診断は病理診断へとつなぐ道筋を照らす羅針盤としての役割を担う。

## 遺伝子異常による分類の意義

第5版では、特定の融合遺伝子（fusion gene）が疾患の定義や診断に直結するようになった。主要な融合遺伝子は以下の通り。

| 腫瘍 | 特徴的な遺伝子異常 |
|------|-------------------|
| 高分化型/脱分化型脂肪肉腫 | MDM2, CDK4増幅 |
| 粘液型脂肪肉腫 | FUS::DDIT3（稀にEWSR1::DDIT3） |
| 滑膜肉腫 | SS18::SSX1/2/4 |
| 胞巣状軟部肉腫 | ASPSCR1::TFE3 |
| 孤立性線維腫 | NAB2::STAT6 |
| Ewing肉腫 | EWSR1::FLI1（85%以上）、EWSR1::ERG |
| 軟部血管線維腫 | AHRR::NCOR2 |
| CIC再構成肉腫 | CIC::DUX4（95%） |
| BCOR遺伝子異常肉腫 | BCOR::CCNB3、BCOR ITD |

遺伝子変異は予後予測・分子標的治療の適応判断に直結するため、画像診断医も組織型と遺伝子異常の対応を把握することが重要である。
