## 概要

悪性末梢神経鞘腫瘍（malignant peripheral nerve sheath tumor; MPNST）は、末梢神経のSchwann細胞または神経周膜細胞から発生する高悪性度肉腫である。

- **好発年齢**：20〜50歳台
- **好発部位**：四肢深部、体幹（傍脊柱領域、後腹膜）
- **臨床症状**：腫瘤、疼痛、神経症状（感覚障害、運動障害）
- **NF1合併**：約半数が**神経線維腫症1型（NF1）**を合併し、蔓状神経線維腫からの悪性転化として発生

## WHO第5版における位置づけ

『WHO骨軟部腫瘍分類（第5版）』では、MPNSTは末梢神経鞘腫瘍の悪性カテゴリーに分類される。第5版では、悪性に**色素性悪性神経鞘腫瘍（melanotic malignant nerve sheath tumour）**が新たに加わり、**悪性顆粒細胞腫（malignant granular cell tumour）**および**悪性神経周膜腫（malignant perineurioma）**が明記された<Cite refs={[1]} />。

## 画像所見

### MRI

- **T1強調像**：神経に沿った腫瘤、低〜中等度信号
- **T2強調像**：不均一な高信号、神経鞘腫よりは均一性が乏しい
- 造影後：不均一造影効果、内部に壊死性領域

### 良性神経鞘腫・神経線維腫との鑑別<Cite refs={[2]} />

| 項目 | 良性神経原性腫瘍 | MPNST |
|------|------------------|-------|
| サイズ | 通常5cm以下 | **5cm以上**が多い |
| 形態 | 境界明瞭、紡錘形 | 境界やや不明瞭、不整形 |
| 内部 | target sign（特徴的） | 不均一、target sign 消失 |
| 造影効果 | 比較的均一 | 不均一、壊死領域 |
| 周囲浮腫 | 軽度 | 高度 |
| 増大速度 | 緩徐 | 急速 |
| 拡散制限 | 軽度 | **強い拡散制限** |

### NF1患者の経過観察

NF1患者で既存の神経線維腫が**急速に増大**したり、**疼痛が出現**した場合は MPNST への悪性転化を強く疑い、**FDG-PET/CT** が有用（高 SUV を示す）。

## 治療・予後

広範切除が標準。化学療法・放射線治療への感受性は限定的。5年生存率は限局例で50%、NF1合併例ではさらに低い（30%以下）。
