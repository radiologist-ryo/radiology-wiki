## 概要

平滑筋肉腫(leiomyosarcoma; LMS)は、平滑筋への分化を示す悪性間葉系腫瘍である。軟部原発のLMSは全身の平滑筋由来組織から発生し、血管壁由来・子宮原発・消化管原発（GIST除く）などのサブタイプがある。<Cite id={1} />

- **好発年齢**：40〜70歳代
- **性別**：女性にやや多い
- **好発部位**：
  - **後腹膜・腹膜・腸間膜**（下大静脈・卵巣静脈・その分枝）
  - 子宮（産婦人科領域で扱われる）
  - 四肢深部筋内（特に大腿）
  - 皮膚（皮膚平滑筋肉腫）

血管原発例では**下大静脈（IVC）leiomyosarcoma**が知られ、大血管閉塞症状を呈することがある。

---

## 臨床像

多くは進行性に増大する腫瘤として発見されるが、症状は非特異的である。

- **後腹膜型**：腹部膨満、腹痛、体重減少。再発・転移率が高く予後不良
- **大血管原発例（IVC LMS）**：下肢浮腫、Budd-Chiari症候群（肝静脈閉塞）
- **四肢型**：無痛性腫瘤、筋内病変
- **皮膚平滑筋肉腫**：表在性、比較的予後良好

## 画像所見

### CT

CTは後腹膜腫瘤の同定、大血管との関係評価、肺転移検索に有用である。

| 所見 | 特徴 |
|------|------|
| 形態 | 大型の充実性腫瘤、分葉状 |
| 造影前 | 筋と等〜軽度低吸収 |
| 造影効果 | **不均一な造影効果**、中心部に壊死・嚢胞変性 |
| 二次変化 | **出血・壊死・嚢胞変性を高頻度に伴う** |
| 大血管 | IVC LMSでは血管内腔の充満・血管壁の破壊 |
| 転移 | 肺（最多）、肝、骨 |

### MRI

| シーケンス | 所見 |
|-----------|------|
| T1強調像 | 筋と等〜低信号、出血部位は高信号 |
| T2強調像 | 不均一な中等度〜高信号、壊死・嚢胞変性を反映した高信号域 |
| 造影後T1 | 充実成分は造影されるが壊死部は非造影、不均一 |
| DWI/ADC | 充実成分で拡散制限 |

### 発生部位別の特徴

**後腹膜LMS**
- 大型腫瘤（10 cm以上のことが多い）
- 出血・壊死・嚢胞変性が目立つ
- 下大静脈・大動脈との関係が重要

**IVC leiomyosarcoma**
- IVC内腔を占拠する充実性腫瘤
- 腎静脈レベル以下が最多
- 造影CT・MRIで血管内進展範囲を同定
- 上行進展があれば右心房まで評価

**四肢型**
- 筋内病変、境界明瞭
- 出血性変化を伴うことあり
- 他の軟部肉腫（UPS、未分化肉腫）との鑑別が必要

## 鑑別診断

### 後腹膜腫瘤

| 疾患 | 鑑別のポイント |
|------|---------------|
| **脱分化型脂肪肉腫** | 脂肪成分を含む。MDM2増幅 |
| **後腹膜肉腫（UPS等）** | 免疫染色で平滑筋マーカー陰性 |
| **神経原性腫瘍** | 傍脊柱発生、神経との連続性 |
| **GIST** | 消化管由来、KIT陽性 |
| **後腹膜線維症** | びまん性、腫瘤形成なし |

### 大血管原発

| 疾患 | 鑑別のポイント |
|------|---------------|
| **血管肉腫** | より侵襲的、血行性転移早期 |
| **血管壁血栓** | 造影効果なし |
| **腫瘍塞栓（腎細胞癌等）** | 原発腫瘍との連続性 |

## 治療

1. **広範切除**が根治の基本。後腹膜型では近接臓器合併切除を要することが多い
2. IVC LMSでは血管切除・再建（人工血管移植）が必要
3. 化学療法はドキソルビシン単剤または**AI療法**（ドキソルビシン+イホスファミド）<Cite id={3} />
4. 再発・難治例：パゾパニブ、トラベクテジン、エリブリン、ゲムシタビン+ドセタキセル
5. 放射線治療は限定的。四肢例では補助療法として検討

---

## 予後

- **後腹膜型**：5年生存率30〜40%と不良
- **四肢型**：5年生存率50〜70%
- **IVC LMS**：切除可能例で5年生存率約50%
- **皮膚型（皮膚筋層平滑筋肉腫）**：表在性で予後良好

---

## 画像診断のポイント

1. **後腹膜の出血・壊死・嚢胞変性を伴う大型腫瘤** → 平滑筋肉腫を鑑別に挙げる
2. **大血管（特にIVC・卵巣静脈）との連続性**は平滑筋肉腫に特徴的
3. **IVC LMS**では血管内進展範囲の正確な評価が手術計画に必須
4. 脱分化型脂肪肉腫との鑑別：**脂肪成分の有無**が決め手
5. 治療効果判定では**壊死領域の拡大・造影効果の減弱**を含む質的評価が重要
