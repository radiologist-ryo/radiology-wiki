## 概要

胞巣状軟部肉腫（alveolar soft part sarcoma; ASPS）は、**ASPSCR1::TFE3融合遺伝子**を特徴とする非常に稀な悪性軟部腫瘍である。<Cite id={1} />

- **好発年齢**：AYA世代〜若年成人（10〜30歳代）
- **性別**：**女性にやや多い**
- **好発部位**：下肢（特に大腿）の深部軟部組織、次いで体幹・内臓・頭頸部
- **わが国での罹患率**：年齢調整罹患率 10万人当たり0.033人と"超希少がん"<Cite id={2} />

---

## 臨床像

緩徐に増大する**無痛性腫瘤**として自覚されることが多い。腫瘍は血流が非常に豊富で、以下の臨床所見が特徴的である。

- **熱感**・**動脈瘤様拍動**の触知
- **血流雑音**の聴取
- **初診時から遠隔転移を伴う症例**が少なくない（肺・骨・脳）

若年女性の大腿部の腫瘤は良性と軽視されやすく、診断の遅れが問題となる。

## 画像所見

### 単純CT

| 所見 | 特徴 |
|------|------|
| 形態 | 境界比較的明瞭、分葉状 |
| 造影前 | 筋と等〜低吸収 |
| 石灰化 | 稀（小型点状のことあり）|

### 造影CT

**動脈相で強い造影効果**を呈し、しばしば**早期静脈還流**（arteriovenous shunt様）がみられる。腫瘤周囲に発達した脈管構造が認められるのが特徴。血管造影でも豊富な腫瘍血管・早期静脈還流が描出される。

### MRI（診断の主役）

ASPSの画像診断ではMRIが中心的役割を果たす。

| シーケンス | 所見 |
|-----------|------|
| T1強調像 | **筋よりやや高信号**（高蛋白・代謝活性高を反映） |
| 脂肪抑制T1 | 信号は抑制されない（脂肪ではない） |
| T2強調像 | 高信号、**内部および周囲にflow void（線状・点状の無信号）** |
| 脂肪抑制T2 | 高信号、flow voidが明瞭 |
| 造影後T1 | 全体が比較的強く造影、flow voidに一致する血管構造 |
| DWI/ADC | 充実成分で拡散制限 |

### 特徴的な画像所見

1. **T1強調像で筋より高信号**：他の軟部肉腫では珍しい所見
2. **腫瘍内外のflow void**：豊富な血流を反映
3. **腫瘍周囲の発達した脈管**：動静脈シャント様
4. **強い均一な造影効果**：hypervascular tumor

これらの所見は**血管性腫瘍（血管腫・動静脈奇形）**と誤認されることがあるが、ASPSは造影でびまん性に強く造影される充実性腫瘤であり、血管腫のような腫瘍血管の集合体とは異なる。

---

## 転移パターン

ASPSは**早期血行性転移**が多く、初診時から転移を有する症例が多い。

- **肺転移**（最多、約50%以上）
- **骨転移**
- **脳転移**（ASPSに特徴的、他の軟部肉腫より高頻度）

診断時に**頭部MRI**を含む全身画像評価が推奨される。

---

## 鑑別診断

| 疾患 | 鑑別のポイント |
|------|---------------|
| **血管肉腫** | より侵襲性、皮膚病変多い、高齢者 |
| **血管平滑筋腫** | 良性、境界明瞭、flow void少ない |
| **孤立性線維腫（SFT）** | 境界明瞭、不均一造影、STAT6陽性 |
| **平滑筋肉腫** | 出血・壊死顕著、大血管原発例あり |
| **血管腫・動静脈奇形** | 充実成分少ない、phlebolith（静脈石） |
| **腎細胞癌転移** | 原発癌の既往 |

## 治療

従来、ASPSは殺細胞性抗癌薬や分子標的薬（血管新生阻害薬）への反応性が乏しいとされてきたが、近年の進歩がある。

### 標準治療

1. **広範切除**（R0切除）による局所制御
2. 放射線治療（術前・術後）の併用

### 新規治療（画像診断医にとって重要）

**免疫チェックポイント阻害薬の有効性**が示唆されている<Cite id={3} />。米国第II相試験（n=52）では、部分奏効37%、完全奏効1例を認め<Cite id={4} />、わが国のALBERT試験（n=20）でも2例に完全奏効が報告された。

これらの結果を受け、**米国では2022年12月に、わが国では2025年2月に、胞巣状軟部肉腫に対する免疫チェックポイント阻害薬（アテゾリズマブ）が薬事承認**を取得した。画像診断医は、従来の化学療法とは異なる反応パターン（pseudo-progression を含む）を理解した上で読影する必要がある。

---

## 予後

- 5年生存率：約60%（転移なし）、約20%（遠隔転移あり）
- 緩徐に進行し、**長期生存例もある**
- 初診時転移例でも、免疫療法導入により予後改善が期待される

---

## 画像診断のポイント

1. **AYA世代女性の下肢軟部腫瘤で、T1高信号＋flow void＋強い造影効果** → ASPSを強く疑う
2. 血管性腫瘍・血管奇形との鑑別には**充実成分の造影効果パターン**が重要
3. 診断時に**肺・骨・脳**の全身検索を行う（特に脳MRIを忘れない）
4. 免疫療法下では**pseudo-progression**が起こりうる。治療継続判断にはChoi基準（造影効果・密度変化）を活用
5. 腫瘍サイズ変化だけでなく、**造影効果の減弱・内部壊死**などの質的変化を総合評価
