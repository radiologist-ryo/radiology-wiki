## 概要

漿液性嚢胞腺腫（serous cystadenoma）は卵巣上皮性腫瘍の中で最も頻度が高い良性腫瘍であり、卵巣嚢胞性腫瘍全体の約20〜30%を占める。腫瘍内腔は卵管上皮に類似した漿液性上皮で覆われ、淡黄色〜淡青色の漿液性内容液を含む。

- **分類**：WHO分類 上皮性腫瘍 > 漿液性腫瘍 > 良性
- **悪性度**：良性（境界悪性の漿液性境界悪性腫瘍 / serous borderline tumor と区別が重要）

---

## 疫学

- 全年齢層に発生するが、**30〜50歳代**に好発
- 卵巣良性腫瘍の中で最多
- 両側性は約10〜20%にみられる（粘液性嚢胞腺腫より高率）
- 悪性転化率は低いが、境界悪性腫瘍との鑑別が臨床上重要

---

## 病理

- **肉眼所見**：単房性または多房性の嚢胞。壁は薄く平滑。内容液は漿液性（水様透明〜淡黄色）。
- **組織所見**：嚢胞内腔は単層の扁平〜立方〜円柱上皮（線毛細胞を含む）で覆われる。細胞異型なし。
- 砂粒体（psammoma body）を伴うことがある。

---

## 画像所見

### 超音波（US）

| 所見 | 特徴 |
|------|------|
| 嚢胞壁 | 薄く平滑（壁厚 < 3 mm） |
| 内部エコー | 無エコー（漿液性内容） |
| 隔壁 | 細い隔壁を有することがある（多房性） |
| 充実成分 | 原則なし |
| 血流 | 壁・隔壁への血流シグナルなし〜乏しい |

- 単房性の薄壁嚢胞が典型像
- 後方エコー増強あり

### CT

| 所見 | 特徴 |
|------|------|
| 形態 | 薄壁の単房〜多房性嚢胞 |
| CT値 | 内容液は水に近い低吸収（0〜20 HU） |
| 壁・隔壁 | 菲薄・平滑、造影効果は軽微〜なし |
| 充実成分 | なし |
| 石灰化 | 砂粒体による点状石灰化を伴うことがある |

### MRI

| 所見 | 特徴 |
|------|------|
| T1強調像 | 低信号（漿液性内容） |
| T2強調像 | 均一高信号 |
| 造影 | 壁・隔壁のみ淡く造影、充実成分は造影されない |
| DWI | 拡散制限なし |

:::note
**境界悪性・悪性との鑑別に重要な所見**  
充実成分・乳頭状突起（papillary projection）・厚い隔壁・腹水が存在する場合は悪性を疑う。
:::

## 鑑別診断

| 疾患 | 鑑別のポイント |
|------|---------------|
| **漿液性境界悪性腫瘍**（serous borderline tumor） | 乳頭状突起を認める。壁不整。確定は病理。 |
| **漿液性嚢胞腺癌**（serous cystadenocarcinoma） | 充実成分・腹水・腹膜播種を伴う。CA125高値。 |
| **粘液性嚢胞腺腫**（mucinous cystadenoma） | 多房性が多い。内容液がより高CT値（粘液性）。T1で高信号のloculiを混じることがある。通常大型。 |
| **機能性嚢胞**（卵胞嚢胞・黄体嚢胞） | 経過で消失することが多い（通常6〜8週で縮小）。壁は薄く小型（< 5 cm）。 |
| **子宮内膜症性嚢胞**（チョコレート嚢胞） | T1で高信号（出血性内容）。T2でshading sign。両側性あり。 |
| **成熟嚢胞性奇形腫**（皮様嚢腫） | 脂肪成分・毛髪・石灰化を含む。T1で高信号。 |
| **傍卵巣嚢胞**（paraovarian cyst） | 卵巣とは独立して存在。卵巣は別に確認可能。 |

---

## IOTA分類（超音波）

IOTA（International Ovarian Tumor Analysis）の単純ルール（simple rules）による評価：

- **M所見なし・B所見あり** → 良性の可能性が高い
  - B1: 単房性嚢胞
  - B2: 充実成分あっても最大径 ≤ 7 mm
  - B3: 音響陰影あり
  - B4: 滑らかな多房性腫瘤（径 ≤ 100 mm）
  - B5: 血流なし

典型的な漿液性嚢胞腺腫はB1またはB4に該当することが多い。

---

## 管理方針（参考）

- 典型的な良性嚢胞所見（薄壁・単房・無血流・充実成分なし・小径）であれば経過観察可
- 腫瘍径が大きい場合（> 5〜7 cm）や増大傾向がある場合は外科的切除を考慮
- 閉経後女性では悪性リスクが上昇するため、より積極的な精査・切除が推奨されることがある
