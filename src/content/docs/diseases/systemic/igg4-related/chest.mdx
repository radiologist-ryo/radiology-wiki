## 概要

全身疾患としてのIgG4関連疾患(IgG4-related disease; IgG4-RD)の疾患概念が確立して以来20年近くが経過し、各臓器病変の画像所見も典型像については認識されてきた<Cite id={1} />。一方で、胸部病変は他臓器病変と比較して診断が難しい領域であり、特に肺病変や後腹膜線維症ではマネジメントに難渋する症例が多い。

IgG4-RDの胸腔内病変としては、肺病変、大動脈/冠動脈周囲病変、傍椎体病変、胸膜/心膜病変が挙げられる。本稿では肺病変と傍椎体病変を中心に解説する。重要なポイントは、IgG4-RDがリンパ路に沿った病変形成を特徴とし、肺病変では肺内リンパ路(気管支血管束、小葉間隔壁)に沿った病変形成を呈すること、そして傍椎体病変は部位や画像所見が特徴的でIgG4-RDを疑う際に必ず確認すべき所見であることである。

## 画像所見

### 肺病変

IgG4-RDの肺病変は当初は間質性肺炎と炎症性偽腫瘍のどちらかと報告されたが<Cite id={2} /><Cite id={3} /><Cite id={4} />、症例蓄積により多彩な像を呈することが明らかとなった<Cite id={2} />。病理組織所見は、リンパ球・形質細胞浸潤、線維化、閉塞性静脈炎からなり、浸潤形質細胞は高率にIgG4陽性となる<Cite id={2} /><Cite id={3} />。肺病変では肺動脈にも閉塞性静脈炎を呈することが知られている。

病変は肺胞隔壁、気管支血管束周囲間質、小葉間隔壁などの広義間質を主座に形成され、これは肺内リンパ路の分布に一致する<Cite id={2} /><Cite id={3} />。肺病変はリンパ路病変として理解できる。

CTではすりガラス影(GGO)、気管支血管束の腫大、小葉間隔壁の肥厚が高頻度に観察され、通常はいくつかの所見が混在してみられる<Cite id={2} />。半数程度に縦隔・肺門部リンパ節腫大を伴う。これらの所見に加え、結節や腫瘤形成を呈するものもある。井上らはCT所見・鑑別すべき疾患に注目して4タイプに分類している<Cite id={2} />。

#### solid nodular type

これまで炎症性偽腫瘍として報告されてきた症例が相当する。画像上は単発性の結節/腫瘤形成を呈し、原発性肺癌が疑われ切除後に診断される。腫瘤辺縁部にスピキュラを伴うことがあり、画像上は原発性肺癌との鑑別が困難。病理組織学的には病変内に多数のリンパ球・形質細胞浸潤、線維化が生じ、既存の肺胞隔壁などの構造を確認できないことが多い。画像のみでIgG4-RDと診断することは困難であり、IgG4-RD経過中に原発性肺癌を併発する症例も存在するため、病理組織学的診断が必要である。

#### round-shaped GGO type

すりガラス影が限局性にみられるタイプで、頻度は少ない。高分化腺癌や細気管支肺胞上皮癌(BAC)との鑑別を要する。診断にはvideo-assisted thoracic surgery(VATS)もしくは開胸下での病理組織学的検索が必要。

#### alveolar interstitial type

いわゆる間質性肺炎として報告されてきた病変。画像上は下葉優位のびまん性すりガラス影に加え、軽度の気管支牽引性拡張を伴う。肺胞腔にも軽度の牽引性拡張を伴い嚢状を呈することがあるが、蜂巣肺を呈する頻度は低い。画像上・病理組織学上はnon-specific interstitial pneumonia(NSIP)パターンを呈することが多い。確定診断には経気管支鏡的生検(TBLB)もしくはVATS下生検が必要となる。

#### bronchovascular type

最も高頻度にみられるタイプで、IgG4関連肺疾患の典型像と考えられる。画像上は気管支血管束の腫大、小葉間隔壁の肥厚が主な所見で、小葉中心性の小結節を伴うことがある。IgG4-RDが上皮に病変を形成することは稀であることを反映し、気管支血管束の腫大部においても罹患部気管支内腔は平滑に保たれることが多いことも特徴。通常、両側肺野にみられる。

本タイプの単独病変では確定診断に病理組織学的検索が必要だが、他部位のIgG4-RDが確定している症例にみられた際にはIgG4-RDの肺病変としてよい。鑑別診断はmulticentric Castleman disease(MCD)、サルコイドーシス、癌性リンパ管症など、広義間質を病変主座とする疾患。特にMCDは画像所見が類似する上に、血清IgG4高値や組織中形質細胞のIgG4陽性となる症例が存在し、重要な鑑別疾患である<Cite id={5} /><Cite id={6} />。ステロイド治療反応性・予後も異なるため、血清IgG4のみならずIL-6、ヘモグロビン、血小板、CRPなどの臨床情報を加味して慎重に判断する<Cite id={6} />。

#### その他の肺病変

器質化肺炎(OP)パターンを呈する症例も知られる<Cite id={7} />。気管支周囲の均等影・コンソリデーションを呈し、周囲にすりガラス影を伴う。solid nodular typeのバリエーションとして理解できる。胸膜にも病変を形成し胸膜肥厚として捉えられ、稀に胸水を伴う<Cite id={4} />。

### 縦隔病変・動脈周囲病変

縦隔・肺門部リンパ節腫大に加え、動脈周囲病変、傍椎体病変、心膜病変などが生じる。胸部大動脈や冠動脈といった比較的大きな動脈周囲に病変を形成するものは、腹部大動脈周囲に生じる病変のスペクトラムとして扱われるようになった。

### 傍椎体病変

傍椎体病変はIgG4-RDの全身病変の中でも変わった病変で、椎体周囲に病変を形成する。単独病変でみられる頻度は低く他臓器病変の合併率が高い。**好発部位が非常に特徴的で、下部胸椎の右側に好発する**<Cite id={8} />。上位胸椎や椎体左側にみられることは稀で、頸椎や腰椎病変は経験がなく文献的報告もない。

横断像では椎体右側縁〜腹側にかけての板状軟部影としてみられる。冠状断像でみると数椎体にわたり頭尾方向に長く形成されることもある。他のIgG4-RDと同様に炎症細胞浸潤・線維化を主体とするため、ダイナミックCT/MRIでは漸増性の造影パターンを示し、静脈相では内部が均一に増強される。椎体周囲に近接してみられるが、椎体や椎間板への炎症波及はみられず骨破壊もない。動脈相では病変内を肋間動脈が貫通する所見もみられ、巻き込まれた肋間動脈に狭窄や閉塞はみられない。

本病変は後腹膜線維症と同様、特定臓器(膵臓・唾液腺など)には分類されず、ターゲットとする解剖学的構造は不明である。著者らは特異的病変分布とIgG4-RDの病変分布がリンパ管に親和性があることから、胸管(分枝を含む)周囲に病変を形成しているのではないかと推測している<Cite id={8} />。

下部胸椎右側に好発するという特徴から、腹部CT撮影内に含まれることがある。膵病変・胆管病変・腎病変などの精査の際には傍椎体病変にも注意を払うことが重要である。本病変はIgG4-RDで疾患特異性が高く、他臓器にIgG4-RDを疑う病変がある際には診断に有用である。

## 鑑別診断

### 肺病変の鑑別

- **原発性肺癌**: solid nodular typeとの鑑別困難、病理学的診断必須
- **高分化腺癌・BAC**: round-shaped GGO typeとの鑑別
- **特発性間質性肺炎(NSIP/UIP)**: alveolar interstitial typeとの鑑別
- **Multicentric Castleman disease (MCD)**: bronchovascular typeと類似、血清IgG4も上昇しうる
- **サルコイドーシス**: リンパ路病変として類似
- **癌性リンパ管症**: 広義間質病変として類似
- **悪性リンパ腫**: 単独病変では鑑別必要

### 傍椎体病変の鑑別

- **悪性リンパ腫**: 重要な鑑別疾患、単独病変ではCTガイド下生検が必要
- **転移性腫瘍**: 椎体破壊を伴うことが鑑別点
- **後腹膜線維症の胸部進展**: 腹部病変との連続性

## 胸部病変の診断における注意点

胸部病変は他臓器病変と比較して診断が難しい。原因として、肺病変の画像所見が多彩であること、組織採取が困難であること、bronchovascular typeを除き悪性腫瘍から良性疾患まで鑑別が多岐にわたること、他疾患でも免疫染色でIgG4陽性となる病態が多数含まれること、少量の病理組織での診断確定が困難であることが挙げられる。

現時点で画像所見からIgG4-RDを強く疑いうるのはbronchovascular typeの病変のみと考えられる。画像診断医としてはbronchovascular typeを肺病変の典型像として理解し、それ以外の病変に関しては血清IgG4値上昇があっても常に疑いの目をもって臨む姿勢が重要である。

2019年のclassification criteria策定時、評価項目に気管支血管束腫大/小葉間隔壁肥厚と傍椎体病変が含まれたのは、その他の肺病変の画像所見の特異性が低いことが一因であった。bronchovascular typeの肺病変と傍椎体病変の画像所見を**"リンパ路病変"**として理解し認識しておくことが、胸部病変の診断には重要である。
