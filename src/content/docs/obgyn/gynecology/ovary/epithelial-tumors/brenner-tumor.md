## 概要

ブレンナー腫瘍（Brenner tumor）は移行上皮（尿路上皮）に類似した細胞からなる卵巣上皮性腫瘍であり、大部分（> 95%）が良性である。境界悪性・悪性ブレンナー腫瘍はまれで、悪性例は移行上皮癌（transitional cell carcinoma）とも称される。しばしば粘液性嚢胞腺腫と混在して発生することが知られる。

---

## 疫学

40〜60歳代の女性に多く発症し、卵巣腫瘍全体の2〜3%を占める。大部分（> 70%）が偶発的に発見される小型の腫瘍であり、片側性がほとんどである。粘液性嚢胞腺腫との混合型が約10〜30%に認められる。

---

## 病理

肉眼所見では小型（通常 < 2 cm）の充実性灰白色腫瘍で、石灰化を高頻度に伴う。組織学的には線維性間質の中に移行上皮様の細胞巣（Walthard細胞巣）を形成し、コーヒー豆核（grooved nuclei）が特徴的である。

---

## 画像所見

### 超音波（US）

超音波では充実性の低エコー腫瘤として描出され、内部に密な石灰化による音響陰影を伴うことが特徴的である。以下に所見をまとめる。

| 所見 | 特徴 |
|------|------|
| 形態 | 小型充実性腫瘤（多くは < 2 cm） |
| エコー | 低エコー |
| 石灰化 | 内部に密な石灰化（音響陰影あり）が特徴 |
| 血流 | 乏しい |

### CT

CTでは充実性腫瘤内に粗大石灰化または緻密石灰化を認めることが特徴的で、腫瘤全体が石灰化により白く描出されることもある。粘液性嚢胞腺腫を合併する場合は充実性成分に隣接する多房性嚢胞が認められる。以下に所見をまとめる。

| 所見 | 特徴 |
|------|------|
| 形態 | 小型充実性（石灰化が特徴的） |
| 石灰化 | 粗大〜緻密な石灰化（腫瘤全体に及ぶこともある） |
| 合併嚢胞 | 粘液性嚢胞腺腫の合併（多房性嚢胞） |
| 造影 | 軽度増強 |

### MRI

MRIでは低T2信号（線維性間質）の充実性腫瘤として描出される。以下に所見をまとめる。

| 所見 | 特徴 |
|------|------|
| T1強調像 | 低〜中等度信号 |
| T2強調像 | 低信号（線維性間質が多い） |
| 石灰化 | 信号欠損として描出 |
| 造影 | 軽度増強 |

---

## 鑑別診断

充実性・石灰化を伴う小型卵巣腫瘤の鑑別として重要である。

| 疾患 | 鑑別のポイント |
|------|---------------|
| **線維腫・莢膜細胞腫** | 充実性、T2低信号（類似）。石灰化は乏しい。 |
| **成熟奇形腫** | 脂肪・毛髪成分。T1高信号。 |
| **卵巣転移** | 原発巣の検索。両側性が多い。 |
| **粘液性嚢胞腺腫** | 多房性嚢胞。充実成分なし（単独の場合）。 |
