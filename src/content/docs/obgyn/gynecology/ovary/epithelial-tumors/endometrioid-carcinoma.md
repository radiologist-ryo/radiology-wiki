## 概要

卵巣類内膜腫瘍（endometrioid tumor）のうち悪性は類内膜癌（endometrioid carcinoma）であり、卵巣癌の10〜15%を占める。子宮内膜症を背景として発生することが多く（約20〜40%）、子宮内膜癌との同時性発生（同時性類内膜癌）を15〜30%に認める。分子病理学的にはARID1A・PTEN・PIK3CA変異が多く、PIK3CA経路の活性化がみられる（Type I癌）。

---

## 疫学

50〜60歳代に多く発症する。子宮内膜症の既往・同時性子宮内膜癌・リンチ症候群（MSI-H）との関連が知られる。Stage I〜IIで発見されることが比較的多く、HGSCよりも予後良好である。

---

## 病理

肉眼所見では充実嚢胞混合性の腫瘤で、内膜症嚢胞（チョコレート嚢胞）内に充実成分が出現した形態を示すことが多い。組織学的には子宮内膜腺癌に類似した管状腺構造を形成し、扁平上皮化生を伴うことがある。子宮内膜症由来の場合はAT（atypical endometriosis）が前駆病変とされる。

---

## 画像所見

### 超音波（US）

超音波では内膜症嚢胞（均一な低エコー・粗い内部エコー）の壁に附随する充実成分として初めて検出されることが多い。以下に所見をまとめる。

| 所見 | 特徴 |
|------|------|
| 嚢胞 | 内膜症嚢胞（低エコー・粗い内部エコー）に充実成分 |
| 充実成分 | 壁在結節状、血流あり |
| 両側性 | 内膜症嚢胞を両側に認めることがある |

### CT

CTでは内膜症嚢胞（高CT値）に充実成分が附随することが特徴的である。以下に所見をまとめる。

| 所見 | 特徴 |
|------|------|
| 嚢胞 | 高CT値（内膜症）に充実成分附随 |
| 充実成分 | 造影増強 |
| 腹水 | 進行例で合併 |
| 子宮病変 | 同時性子宮内膜癌の検索が必要 |

### MRI

MRIはチョコレート嚢胞と充実成分の評価に最も有用なモダリティである。内膜症嚢胞はT1高信号・T2 shading signが特徴的で、その壁に充実成分が付随している所見が悪性化を示唆する。以下に所見をまとめる。

| 所見 | 特徴 |
|------|------|
| T1強調像 | 嚢胞：高信号（内膜症）、充実成分：中等度 |
| T2強調像 | Shading sign＋充実成分（中等度信号） |
| DWI | 充実成分に拡散制限 |
| 造影 | 充実成分が増強 |

---

## 鑑別診断

内膜症嚢胞に充実成分が出現した場合は悪性化を疑い精査を要する。

| 疾患 | 鑑別のポイント |
|------|---------------|
| **明細胞癌** | 内膜症由来。充実成分の形態が異なることあり。 |
| **高異型度漿液性癌（HGSC）** | より大きな充実成分・腹膜播種。TP53変異。 |
| **子宮内膜症嚢胞（良性）** | 充実成分なし。シャドーイング（嚢胞壁の肥厚） |
| **子宮内膜癌の卵巣転移** | 同時性発生との区別は病理で確定。 |
