## 概要

高異型度漿液性癌（high-grade serous carcinoma; HGSC）は卵巣悪性腫瘍の中で最も頻度が高く、卵巣癌全体の70〜80%を占める。近年の分子病理学的研究から、多くのHGSCは卵巣表面上皮ではなく卵管采上皮（fimbriae）を起源とするとされる。TP53変異はほぼ全例に認められ、BRCA1/BRCA2変異を約15〜20%に伴う。診断時にはすでに腹膜播種を来しているStage IIIが多い。

---

## 疫学

50〜70歳代の閉経後女性に多く発症する。家族歴（BRCA変異）、排卵回数の多い女性（未産婦、初経早発、閉経遅延）がリスク因子とされる。一方、経口避妊薬や授乳はリスクを低下させる。CA125は感度70〜80%と診断補助に有用だが特異度は低い。

---

## 病理

肉眼所見では充実性と嚢胞性が混在する複雑な構造を呈し、壊死・出血を伴うことが多い。組織学的には高度な核異型と活発な核分裂像を示す漿液性上皮細胞からなり、間質へ浸潤する。TP53変異は免疫染色でほぼ全例に陽性または陰性（null型）を示す。

---

## 画像所見

### 超音波（US）

超音波では大型の充実嚢胞混合性腫瘤として描出される。充実成分は不均一で血流が豊富であり、腹水を伴うことが多い。以下に所見をまとめる。

| 所見 | 特徴 |
|------|------|
| 腫瘤形態 | 大型充実嚢胞混合性 |
| 充実成分 | 不均一、厚い不整壁 |
| 血流 | 充実成分に豊富 |
| 腹水 | 高率で合併 |

### CT

CTでは充実嚢胞混合性の付属器腫瘤に加え、腹膜播種・大網浸潤（omental cake）・リンパ節腫大が特徴的である。腹膜播種は小網・腸間膜・横隔膜腹膜に多発結節として認められる。以下に所見をまとめる。

| 所見 | 特徴 |
|------|------|
| 腫瘤 | 充実嚢胞混合性、両側性あり（30〜40%） |
| 腹膜播種 | 多発小結節、大網浸潤（omental cake） |
| リンパ節 | 骨盤・傍大動脈リンパ節腫大 |
| 腹水 | 中〜大量の腹水 |
| 胸水 | 横隔膜播種に伴う胸水 |

### MRI

MRIでは充実成分がT2強調像で中等度信号、DWIで拡散制限を示す。MRIは腹膜播種の検出や手術可能性の評価（腸管・血管への浸潤）に有用である。以下に所見をまとめる。

| 所見 | 特徴 |
|------|------|
| T2強調像 | 充実成分：中等度不均一信号 |
| DWI | 充実成分・腹膜播種に拡散制限 |
| 造影 | 充実成分が著明に増強 |

## 鑑別診断

| 疾患 | 鑑別のポイント |
|------|---------------|
| **低異型度漿液性癌（LGSC）** | 境界悪性腫瘍に隣接、腹膜に砂粒体多発。TP53変異なし。 |
| **類内膜癌** | 子宮内膜症に合併多い。ARID1A変異。 |
| **明細胞癌** | 内膜症嚢胞内に充実成分。特に日本で多い。 |
| **転移性腫瘍（Krukenberg）** | 両側性、原発巣の検索が重要。 |
| **卵管癌** | 解剖学的には区別困難。病理で確定。 |
