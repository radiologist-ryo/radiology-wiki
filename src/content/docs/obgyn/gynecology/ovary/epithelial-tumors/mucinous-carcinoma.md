## 概要

卵巣原発の粘液性癌（mucinous carcinoma）は卵巣悪性上皮性腫瘍の3〜5%を占める比較的まれな腫瘍である。消化管・膵臓・子宮頸部からの転移性粘液性腫瘍との鑑別が実臨床上の最大の課題であり、卵巣に粘液性悪性腫瘍を認めた場合は原発巣検索が必須である。原発性の場合は粘液性嚢胞腺腫・境界悪性腫瘍からの悪性転化経路（Type I癌）が推定される。

---

## 疫学

20〜50歳代に発生し、原発性の場合は大部分が片側性でStage Iで発見されることが多い。両側性（> 10%）や腹膜播種を認める場合は転移性を強く疑う。CA125は上昇しないことが多く、CEAやCA19-9が上昇する傾向がある。

---

## 病理

肉眼所見では粘液性嚢胞腺腫に類似した大型多房性嚢胞であるが、充実成分・壁肥厚・乳頭状突起を伴う。組織学的には腸型腺癌に類似した粘液産生細胞からなり、膨張性浸潤（expansile invasion）と浸潤性浸潤（infiltrative invasion）に分類される。

---

## 画像所見

### 超音波（US）

超音波では大型の多房性嚢胞に充実成分・壁肥厚を認める。以下に所見をまとめる。

| 所見 | 特徴 |
|------|------|
| 形態 | 大型多房性（充実成分・壁肥厚あり） |
| 充実成分 | 結節状・乳頭状に増強される |
| 腹水 | 進行例で合併 |

### CT

CTでは多房性嚢胞の一部に厚い壁・充実成分・造影増強を認める。腹腔内に粘液性腹水（mucinous ascites）を伴う場合は腹膜偽粘液腫を合併・または転移性を疑う。以下に所見をまとめる。

| 所見 | 特徴 |
|------|------|
| 形態 | 大型多房性、壁肥厚・充実成分あり |
| CT値 | 房ごとに異なる粘液性内容（高CT値） |
| 充実成分 | 造影増強あり |
| 腹水 | 粘液性腹水（高CT値）を伴うことあり |

### MRI

MRIでは多房性嚢胞の充実成分がT2中等度信号・DWI拡散制限を示す。以下に所見をまとめる。

| 所見 | 特徴 |
|------|------|
| T1強調像 | 粘液性内容：中等度〜高信号 |
| T2強調像 | 充実成分：中等度信号 |
| DWI | 充実成分に拡散制限 |
| 造影 | 充実成分が増強 |

---

## 鑑別診断

両側性・小型・腹膜播種が目立つ場合は転移性を強く疑う。

| 疾患 | 鑑別のポイント |
|------|---------------|
| **転移性粘液性腫瘍（Krukenberg）** | 両側性、原発巣あり（胃・大腸・膵臓・子宮頸部）。 |
| **粘液性嚢胞腺腫** | 充実成分なし。壁平滑。 |
| **粘液性境界悪性腫瘍** | 間質浸潤なし。病理で確定。 |
| **腹膜偽粘液腫** | 虫垂原発。腹腔内全体に粘液性腫瘍。 |
