## 概要

粘液性嚢胞腺腫（mucinous cystadenoma）は卵巣上皮性腫瘍の中で2番目に多い良性腫瘍であり、卵巣嚢胞性腫瘍全体の約15〜25%を占める。腫瘍内腔は粘液産生上皮（子宮頸管内膜型または腸型）で覆われ、粘液性内容液を含む。多房性で大型となる傾向があり、直径20 cmを超えることもある。

---

## 疫学

30〜50歳代に好発し、漿液性嚢胞腺腫と比較して発症年齢はやや高い傾向がある。両側性は約5%と少なく、ほぼ一側性に発生する。悪性転化（粘液性嚢胞腺癌）のリスクは低いが、大型・多房性の場合は注意が必要である。卵巣奇形腫との合併例も報告される。

---

## 病理

肉眼所見では大型の多房性嚢胞を形成し、房ごとに異なる色調の粘液性内容物（淡黄色〜緑色、ゼリー状）を含む。嚢胞壁は薄く平滑で、充実成分は原則として認めない。組織学的には核異型のない粘液産生の単層円柱上皮で覆われ、腸型（goblet cell含む）と子宮頸管内膜型に分類される。

---

## 画像所見

### 超音波（US）

超音波では多房性の嚢胞性腫瘤として描出され、各房の内容物の性状が異なるために内部エコーが房ごとに異なることがある（heterogeneous multilocular appearance）。以下に所見をまとめる。

| 所見 | 特徴 |
|------|------|
| 形態 | 大型多房性嚢胞（最多30 cm超） |
| 内部エコー | 房ごとに異なる（無エコー〜低エコー混在） |
| 隔壁 | 細い隔壁（多数） |
| 充実成分 | 原則なし |
| 血流 | 壁・隔壁に乏しい |

### CT

CTでは水様〜粘液性の様々な低吸収値を示す多房性嚢胞として描出される。漿液性と比較してCT値がやや高い（15〜30 HU）ことが特徴的で、房ごとにCT値が異なる（stained glass pattern）。以下に所見をまとめる。

| 所見 | 特徴 |
|------|------|
| 形態 | 多房性大型嚢胞（一側性） |
| CT値 | 漿液性より高い（15〜30 HU）、房ごとに異なる |
| 壁・隔壁 | 薄く平滑、造影効果は軽微〜なし |
| 充実成分 | なし |

### MRI

MRIでは各房の粘液濃度の違いによりT1・T2信号強度が房ごとに異なる所見（multilocular appearance with variable signal）が特徴的である。以下に所見をまとめる。

| 所見 | 特徴 |
|------|------|
| T1強調像 | 低〜中等度信号（濃い粘液ではやや高信号） |
| T2強調像 | 高信号主体、房ごとに信号が異なる |
| 造影 | 壁・隔壁のみ軽度増強 |
| 特徴 | 各房の信号が不均一（stained glass pattern） |

## 鑑別診断

| 疾患 | 鑑別のポイント |
|------|---------------|
| **漿液性嚢胞腺腫** | 単房〜少房性。均一な水様信号。小型が多い。 |
| **粘液性嚢胞腺癌** | 充実成分・壁肥厚・腹水を伴う。 |
| **成熟奇形腫（多房性）** | 脂肪・石灰化成分。T1高信号。 |
| **転移性腫瘍（Krukenberg）** | 両側性。胃癌などの原発巣あり。 |
| **腹膜偽粘液腫（PMP）** | 虫垂や大腸原発の粘液腫。腹腔内ゼラチン状腫瘍。 |
