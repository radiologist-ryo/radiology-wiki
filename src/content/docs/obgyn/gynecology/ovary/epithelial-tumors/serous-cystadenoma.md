## 概要

漿液性嚢胞腺腫（serous cystadenoma）は卵巣上皮性腫瘍の中で最も頻度が高い良性腫瘍であり、卵巣嚢胞性腫瘍全体の約20〜30%を占める。腫瘍内腔は卵管上皮に類似した漿液性上皮で覆われ、淡黄色〜淡青色の漿液性内容液を含む。WHO分類では上皮性腫瘍 > 漿液性腫瘍 > 良性に位置づけられる。臨床上は良性であるが、境界悪性の漿液性境界悪性腫瘍（serous borderline tumor）との鑑別が重要である。

---

## 疫学

全年齢層に発生しうるが、30〜50歳代に好発し、卵巣良性腫瘍の中で最多を占める。両側性は約10〜20%にみられ、単房性の粘液性嚢胞腺腫と比較して両側発生の頻度が高い。悪性転化率そのものは低いものの、境界悪性腫瘍との画像上の鑑別が臨床的に重要な課題となる。

---

## 病理

肉眼所見では単房性または多房性の嚢胞を形成し、壁は薄く平滑で、内容液は水様透明〜淡黄色の漿液性である。組織学的には嚢胞内腔が単層の扁平〜立方〜円柱上皮（線毛細胞を含む）で覆われ、細胞異型を認めない。砂粒体（psammoma body）を伴うことがあり、壁の点状石灰化として画像に反映されることがある。

---

## 画像所見

### 超音波（US）

超音波では単房性または少房性の無エコー嚢胞として描出される。壁は菲薄で平滑であり、後方エコー増強を伴うことが特徴的である。充実成分や乳頭状突起がなければ典型的な良性嚢胞として経過観察が可能であり、悪性を積極的に示唆する所見に乏しい。以下に所見をまとめる。

| 所見 | 特徴 |
|------|------|
| 嚢胞壁 | 薄く平滑（壁厚 < 3 mm） |
| 内部エコー | 無エコー（漿液性内容） |
| 隔壁 | 細い隔壁を有することがある（多房性） |
| 充実成分 | 原則なし |
| 血流 | 壁・隔壁への血流シグナルなし〜乏しい |

### CT

CTでは水に近い低吸収値（0〜20 HU）の嚢胞性腫瘤として描出され、造影剤による有意な増強効果を示す充実成分は認めない。壁や隔壁への軽微な造影は許容されるが、結節状の造影増強が認められる場合は境界悪性や悪性を疑う必要がある。砂粒体による微細石灰化が壁に散在することがあり、腫瘍の長期存在を示唆する所見として知られる。以下に所見をまとめる。

| 所見 | 特徴 |
|------|------|
| 形態 | 薄壁の単房〜多房性嚢胞 |
| CT値 | 内容液は水に近い低吸収（0〜20 HU） |
| 壁・隔壁 | 菲薄・平滑、造影効果は軽微〜なし |
| 充実成分 | なし |
| 石灰化 | 砂粒体による点状石灰化を伴うことがある |

### MRI

MRIでは漿液性内容液を反映し、T1強調像で低信号、T2強調像で均一な高信号を呈する。T1強調像で高信号を示す場合は出血性内容（チョコレート嚢胞）や粘液性成分との鑑別が重要となる。DWIで拡散制限を認めず、充実成分が造影されない場合は漿液性嚢胞腺腫に矛盾しない所見であり、MRIは良悪性鑑別において最も有用なモダリティとされる。以下に所見をまとめる。

| 所見 | 特徴 |
|------|------|
| T1強調像 | 低信号（漿液性内容） |
| T2強調像 | 均一高信号 |
| 造影 | 壁・隔壁のみ淡く造影、充実成分は造影されない |
| DWI | 拡散制限なし |

---

## 鑑別診断

充実成分・乳頭状突起（papillary projection）・厚い隔壁・腹水のいずれかが存在する場合は良性漿液性嚢胞腺腫よりも悪性または境界悪性を積極的に疑う必要がある。

| 疾患 | 鑑別のポイント |
|------|---------------|
| **漿液性境界悪性腫瘍**（serous borderline tumor） | 乳頭状突起を認める。壁不整。確定は病理。 |
| **漿液性嚢胞腺癌**（serous cystadenocarcinoma） | 充実成分・腹水・腹膜播種を伴う。CA125高値。 |
| **粘液性嚢胞腺腫**（mucinous cystadenoma） | 多房性が多い。内容液がより高CT値（粘液性）。T1で高信号のloculiを混じることがある。通常大型。 |
| **機能性嚢胞**（卵胞嚢胞・黄体嚢胞） | 経過で消失することが多い（通常6〜8週で縮小）。壁は薄く小型（< 5 cm）。 |
| **子宮内膜症性嚢胞**（チョコレート嚢胞） | T1で高信号（出血性内容）。T2でshading sign。両側性あり。 |
| **成熟嚢胞性奇形腫**（皮様嚢腫） | 脂肪成分・毛髪・石灰化を含む。T1で高信号。 |
| **傍卵巣嚢胞**（paraovarian cyst） | 卵巣とは独立して存在。卵巣は別に確認可能。 |

---

## IOTA分類（超音波）

IOTA（International Ovarian Tumor Analysis）の単純ルール（simple rules）による評価：

- **M所見なし・B所見あり** → 良性の可能性が高い
  - B1: 単房性嚢胞
  - B2: 充実成分あっても最大径 ≤ 7 mm
  - B3: 音響陰影あり
  - B4: 滑らかな多房性腫瘤（径 ≤ 100 mm）
  - B5: 血流なし

典型的な漿液性嚢胞腺腫はB1またはB4に該当することが多い。

---

## 管理方針（参考）

典型的な良性嚢胞所見（薄壁・単房・無血流・充実成分なし・小径）であれば経過観察が許容される。腫瘍径が大きい場合（> 5〜7 cm）や増大傾向がある場合は外科的切除を考慮し、閉経後女性では悪性リスクが上昇するため、より積極的な精査・切除が推奨されることがある。
