## 概要

成熟嚢胞性奇形腫（mature cystic teratoma; MCT）は卵巣胚細胞性腫瘍の中で最も多く、良性腫瘍である。外胚葉・中胚葉・内胚葉の3胚葉由来組織（皮膚・毛髪・歯・脂肪・神経組織など）を含むことから皮様嚢腫（dermoid cyst）とも呼ばれる。まれに（0.5〜2%）悪性転化（多くは扁平上皮癌）をきたす。

---

## 疫学

生殖年齢の若い女性（10〜40歳代）に多く発症し、卵巣腫瘍全体の10〜20%を占める。両側性は約10〜15%に認められる。超音波で偶発的に発見されることが多い。腫瘍茎捻転の発生率が高く（約3〜16%）、急性腹症として手術適応となることがある。

---

## 病理

肉眼所見では嚢胞内に脂肪・皮脂腺分泌物・毛髪が充満し、Rokitansky隆起（充実性結節部）を形成する。Rokitansky隆起内に歯・骨・神経組織などが含まれる。組織学的には成熟した3胚葉組織が認められる。

---

## 画像所見

### 超音波（US）

超音波では脂肪性内容による高エコー部分と、Rokitansky隆起の強エコー（骨・歯に伴う音響陰影）が特徴的である。dermoid meshと呼ばれる線状の強エコーパターンも認められる。以下に所見をまとめる。

| 所見 | 特徴 |
|------|------|
| 内部エコー | 不均一高エコー（脂肪・毛髪） |
| Rokitansky隆起 | 強エコー＋音響陰影（歯・骨） |
| Dermoid mesh | 線状高エコーパターン（毛髪） |
| 後方 | 後方エコー増強またはシャドーイング |

### CT

CTでは脂肪成分による著明な低吸収域（−20〜−100 HU）が特徴的であり、歯・骨の高吸収域とともに認められる。Rokitansky隆起は軟部組織密度で造影増強を示す。以下に所見をまとめる。

| 所見 | 特徴 |
|------|------|
| 脂肪成分 | 著明低吸収（−20〜−100 HU）が診断的 |
| 歯・骨 | 高吸収（高CT値） |
| 液・脂肪境界面 | 脂肪-液面形成（fat-fluid level） |
| 石灰化 | Rokitansky隆起内 |

### MRI

MRIでは脂肪成分によるT1高信号が特徴的であり、脂肪抑制像で信号が著明に低下する（脂肪との確認に有用）。以下に所見をまとめる。

| 所見 | 特徴 |
|------|------|
| T1強調像 | 高信号（脂肪）が診断的 |
| T2強調像 | 高信号（脂肪）、Rokitansky隆起は中等度 |
| 脂肪抑制 | 信号著明低下（脂肪確認） |
| 化学シフトアーティファクト | 脂肪-水界面にインドウ状アーティファクト |

---

## 鑑別診断

T1高信号の卵巣嚢胞は脂肪性・出血性・粘液性の3つを考える。

| 疾患 | 鑑別のポイント |
|------|---------------|
| **子宮内膜症嚢胞** | T1高信号（出血）。脂肪抑制で低下しない。T2 shading sign。 |
| **粘液性嚢胞腺腫** | T1中等度〜高信号。脂肪抑制で低下しない。多房性。 |
| **未熟奇形腫** | 脂肪＋充実性成分（神経組織）。大型。 |
| **脂肪肉腫** | まれ。充実成分主体。 |
