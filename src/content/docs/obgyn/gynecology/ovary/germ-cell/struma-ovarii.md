## 概要

卵巣甲状腺腫（struma ovarii）は、成熟奇形腫の一亜型として位置づけられる比較的稀な疾患で、腫瘍内に甲状腺組織が主成分（50%以上）を占めるものを指す。

無症候性の甲状腺ホルモン産生例は少なくないとされるが、臨床的に甲状腺機能亢進症を呈する例は頻度が低く、卵巣甲状腺腫と病理学的に診断された症例の5〜15%とされる。

---

## 疫学・臨床

異所性甲状腺の有病率は、健常者では10万〜30万人に1人程度、甲状腺疾患患者では4000〜8000人に1人と報告される。発生部位は圧倒的に舌根部が多く、次に頭頸部（気管、顎下部、口蓋扁桃など）、腋窩、消化管、副腎、生殖器など多岐にわたる。単施設47例52病変の検討では、頭頸部領域（40病変、77%）の次に多いのが卵巣（7病変、13.4%）であった。

機能性卵巣甲状腺腫の頻度は稀とされるが、**甲状腺機能亢進症の原因として異所性甲状腺組織が考えられる場合、卵巣甲状腺腫はその原因として決して頻度が低いものではない**。

## 画像所見

### 超音波（US）

充実性腫瘤を認め、内部エコーは不均一である。明らかな脂肪成分（dermoid mesh や fat-fluid level）は通常認められないことが、成熟嚢胞性奇形腫との鑑別点となる。血流は豊富なことが多い。

### CT

卵巣に充実性腫瘤を認め、充実部はCTで高吸収を呈することが特徴的である。これは甲状腺組織内のコロイドがヨードを豊富に含むためとされる。明らかな脂肪成分は通常認められない。

| 所見 | 特徴 |
|------|------|
| 密度 | 充実部が高吸収（甲状腺コロイド由来） |
| 脂肪成分 | 通常なし（成熟嚢胞性奇形腫との鑑別点） |
| 造影 | 充実成分に明瞭な造影効果 |

### MRI

腫瘤辺縁の充実部はT2強調像で低信号、T1強調像で中等度信号を呈する。充実成分には明瞭な造影効果が認められる。明らかな脂肪成分が認められないことが、典型的な成熟嚢胞性奇形腫との主な鑑別点となる。

| 所見 | 特徴 |
|------|------|
| T1強調像 | 充実部：中等度信号 |
| T2強調像 | 充実部：低信号〜中等度信号（甲状腺組織） |
| DWI | 明らかな拡散制限なし |
| 造影 | 充実成分に明瞭な造影効果 |

### 核医学検査

機能性の卵巣甲状腺腫が疑われる場合、Na¹²³I を用いた甲状腺シンチグラフィが診断に有用である。腫瘤に一致して明瞭な集積が認められる。

¹²³I-NaI シンチグラフィでは、機能性卵巣甲状腺腫が活動性である場合、頸部甲状腺への集積はTSH抑制により低下する。背景甲状腺への集積が乏しく、サイログロブリン上昇を伴い、頸部〜縦隔上部に明らかな集積がない場合は、異所性甲状腺ホルモン産生腫瘍を疑う根拠となる。

---

## 診断のフロー

異所性甲状腺ホルモン産生腫瘍を疑った場合、その責任部位を頭頸部〜縦隔のみと考えていると、胸部CTのみを撮影される可能性がある。一方、頻度の観点からも卵巣は確認する必要があり、**全身CT検索や Na¹²³I を用いた全身の甲状腺シンチグラフィが診断に有用**である。

---

## 鑑別診断

| 疾患 | 鑑別のポイント |
|------|---------------|
| **成熟嚢胞性奇形腫** | 脂質（皮脂腺由来）と液面形成を伴う嚢胞性腫瘤、脂肪抑制で信号抑制 |
| **線維腫・ブレンナー腫瘍** | T2強調像で低信号を呈するが、甲状腺シンチで集積なし |
| **甲状腺癌の卵巣転移** | 既往甲状腺癌の有無、両側性、腹膜播種の有無 |
| **悪性転化を伴う卵巣甲状腺腫** | 稀だが報告あり。不整な辺縁、腹水、転移所見 |

---

## 臨床的ポイント

- 卵巣甲状腺腫は稀ながら甲状腺ホルモンを産生し、甲状腺機能亢進症の原因となりうる
- 異所性甲状腺ホルモン産生を疑い、頸部〜縦隔に病変が確認されなかった場合、女性患者で次に考えるべきは卵巣
- 術前に甲状腺機能亢進症を把握しておかないと、術中・術後の甲状腺クリーゼのリスクがある
- 全身CTや Na¹²³I 全身甲状腺シンチグラフィが診断に有用
