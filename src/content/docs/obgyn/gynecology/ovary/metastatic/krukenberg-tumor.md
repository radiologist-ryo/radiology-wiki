## 概要

クルーケンベルグ腫瘍（Krukenberg tumor）は卵巣への転移性腫瘍であり、粘液産生性の印環細胞癌（signet ring cell carcinoma）が組織学的定義である。原発巣として胃癌が最も多く（70〜80%）、次いで大腸癌・膵癌・乳癌・虫垂癌などがある。卵巣転移の経路としてはリンパ行性・血行性・腹膜播種性が考えられているが、詳細は不明である。

---

## 疫学

40〜50歳代の女性に多く、両側性卵巣腫瘤として発見される。卵巣腫瘍全体の1〜2%を占めるが、消化器癌既往の女性では鑑別の優先度が高い。原発巣の発見前に卵巣腫瘤として発症することがあり、原発巣検索が不可欠である。予後は極めて不良（中央生存期間 < 12か月）。

---

## 病理

肉眼所見では充実嚢胞混合性で、断面が多彩な外観（ゼラチン状・粘液性・充実性）を呈する。組織学的には卵巣固有間質（stromal）内に粘液産生の印環細胞が浸潤するパターンが特徴的で、卵巣間質が肉腫様に増殖（desmoplastic stroma）することがある。

---

## 画像所見

### 超音波（US）

超音波では両側性の充実嚢胞混合性腫瘤として描出される。卵巣間質の増殖を反映した充実成分が血流豊富なことがある。以下に所見をまとめる。

| 所見 | 特徴 |
|------|------|
| 両側性 | 高率（> 80%）に両側性 |
| 形態 | 充実嚢胞混合性 |
| 内部 | 不均一（充実・嚢胞混在） |
| 血流 | 充実成分に血流あり |

### CT

CTでは両側性の充実嚢胞混合性卵巣腫瘤として描出される。腹膜播種・大網浸潤・リンパ節転移が合併することが多い。原発巣の検索のために上腹部〜骨盤腔全体を評価する。以下に所見をまとめる。

| 所見 | 特徴 |
|------|------|
| 両側性 | 両側卵巣腫大 |
| 内部 | 充実成分＋多房性嚢胞（不均一） |
| 造影 | 充実成分が増強 |
| 腹膜播種 | 腹膜結節・大網浸潤 |
| 原発巣 | 胃壁肥厚・大腸腫瘤などの検索が必須 |

### MRI

MRIでは卵巣間質の線維化を反映してT2低〜中等度信号の充実成分と多房性嚢胞が混在する外観が特徴的である。以下に所見をまとめる。

| 所見 | 特徴 |
|------|------|
| T1強調像 | 充実成分：中等度信号 |
| T2強調像 | 充実成分：低〜中等度信号（線維化）＋嚢胞成分 |
| DWI | 充実成分に拡散制限 |
| 造影 | 充実成分が増強 |

## 鑑別診断

両側性・充実嚢胞混合性の卵巣腫瘤では転移性を常に念頭に置く。

| 疾患 | 鑑別のポイント |
|------|---------------|
| **原発性卵巣癌（HGSC）** | 漿液性主体。腹膜播種はより顕著。原発巣なし。 |
| **粘液性嚢胞腺腫/腺癌** | 通常一側性。原発巣なし。 |
| **子宮内膜症嚢胞** | T1高信号（出血）。shading sign。通常一側性〜両側性。 |
| **顆粒膜細胞腫** | 内分泌症状あり。インヒビン高値。充実性主体。 |
