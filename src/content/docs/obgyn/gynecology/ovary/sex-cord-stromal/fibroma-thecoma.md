## 概要

卵巣線維腫（fibroma）および莢膜細胞腫（thecoma）は卵巣性索間質性腫瘍の良性腫瘍であり、線維芽細胞または莢膜細胞（内膜細胞）から構成される。線維腫はメイグス症候群（Meigs syndrome：線維腫＋腹水＋胸水）を合併することがあり、腫瘍切除後に腹水・胸水は消失する。莢膜細胞腫はエストロゲンを産生し、子宮内膜増殖症や子宮内膜癌を合併することがある。

---

## 疫学

線維腫は卵巣腫瘍の3〜5%を占め、良性腫瘍である。メイグス症候群は線維腫の1〜3%に合併する。莢膜細胞腫は閉経後女性に多く、エストロゲン産生による内分泌症状（不正出血・子宮内膜肥厚）を呈することがある。線維莢膜細胞腫（fibrothecoma）は両者の移行型として認識されている。

---

## 病理

肉眼所見では線維腫は充実性・灰白色・硬い腫瘍で、多くは5 cm未満である。莢膜細胞腫は黄色調の充実性腫瘍で、脂質に富む莢膜細胞を含む。組織学的には線維腫は密な紡錘形細胞（線維芽細胞）からなり、莢膜細胞腫はlipid-laden theca cellsを含む。

---

## 画像所見

### 超音波（US）

超音波では充実性の低エコー腫瘤として描出される。後方エコーの減弱（音響陰影）を認めることがある。以下に所見をまとめる。

| 所見 | 特徴 |
|------|------|
| 形態 | 充実性（まれに小嚢胞を含む） |
| エコー | 低〜中エコー（後方減弱あり） |
| 血流 | 乏しい〜なし |
| 腹水 | メイグス症候群では腹水・胸水 |

### CT

CTでは充実性低吸収腫瘤（筋肉程度の吸収値）として描出され、造影効果は軽度〜中等度である。以下に所見をまとめる。

| 所見 | 特徴 |
|------|------|
| 形態 | 充実性（辺縁明瞭） |
| CT値 | 筋肉程度の低吸収値 |
| 造影 | 軽度〜中等度の増強 |
| 腹水・胸水 | メイグス症候群では合併 |

### MRI

MRIでは線維性成分を反映してT2強調像で著明な低信号を示す所見が特徴的であり、性索間質性腫瘍の中で最も診断的価値が高い。以下に所見をまとめる。

| 所見 | 特徴 |
|------|------|
| T1強調像 | 中等度低信号 |
| T2強調像 | **著明な低信号**（線維成分が多い）が特徴的 |
| 造影 | 軽度〜中等度の増強 |
| DWI | 拡散制限は通常なし |

---

## 鑑別診断

T2低信号の充実性卵巣腫瘤として、子宮筋腫（広間膜筋腫）との鑑別も重要である。

| 疾患 | 鑑別のポイント |
|------|---------------|
| **ブレンナー腫瘍** | 充実性・石灰化が特徴的。 |
| **子宮筋腫（広間膜）** | 子宮との連続性。T2低信号（類似）。 |
| **顆粒膜細胞腫** | T2中等度〜高信号。内分泌症状あり。 |
| **卵巣線維肉腫** | まれ。急速増大。 |
